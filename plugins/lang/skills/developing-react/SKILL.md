  React 19.x development guide covering internals (rendering, reconciliation, Fiber), design
  patterns (Container/Presenter, HOC, Render Props, Compound Components), state management
  (useState to Zustand/TanStack Query, nuqs/Jotai), error handling (ErrorBoundary, React 19 error

  optimization (react-doctor, React Compiler, memoization), UI animation patterns (CSS transitions,
  easing), React Testing Library (RTL queries, interactions, TDD patterns), and Storybook (CSF3,
  interaction testing, a11y). Also covers React Router, auth/RBAC, i18n, OpenAPI, and production
  architecture. Use when package.json contains 'react' (without 'next'), or when working on
  React-specific concerns in any framework.
when_to_use: >-

  testing-e2e-with-playwright. For general testing methodology, use testing-code. For backend
  APIs, use developing-fullstack-javascript.
