## 1. ビジュアルコミュニケーションの原理

グラフィックデザインが伝えられる情報は、文字・記号による**言語情報**だけではない。色・形・図・写真などの**非言語情報**も含まれており、両者を有機的に組み合わせることを「視覚伝達（ビジュアルコミュニケーション）」という。

デザイナーの役割は「要素間の関係性を定義すること」。要素を演出家のように組み合わせ、主役・脇役・エキストラの役割を明確にする。何を入れ、何を除くかの判断もデザイナーの仕事。

**効果的なレイアウトの原則**:
- 要素の優先順位を決める（主菜・副菜・その他）
- 集中と分散、緊張と緩和の対比バランスを意識する
- 「幕の内弁当」より「単品弁当」—メリハリのある構成を目指す

## 2. 構図パターン

### 基本分割パターン

| 構図 | 特徴 |
|------|------|
| **二等分割** | 上下/左右を均等に分割。拮抗した関係感。写真×色面などに適する |
| **三等分割** | 二等分割より緊張感が弱まる。2:1の関係にも変化可能 |
| **四等分割** | 無機的・フラットな印象。1面を空けるとメリハリが生まれる |
| **黄金分割** | 約1:1.618の比率。トリミング・配置の基準点に活用 |

### 視覚的構図パターン

| 構図 | 効果 |
|------|------|
| **円・放射** | 中心が明快。アイキャッチ効果が高い |
| **三角形** | 安定した視線誘導。底辺下配置で版面が落ち着く |
| **線対称（シンメトリー）** | 誘目性と安定感を兼ね備える。格調ある印象 |
| **点対称** | 対角線方向の視線移動。印象に残りやすい |
| **軸** | 中央配置→堂々と、端配置→緊張感ある空間に |
| **反復** | 類似要素のリズム。アクセントで単調さを防ぐ |
| **孤立** | 要素を孤立させ周囲にホワイトスペース。強調＋空間性 |
| **レイヤー** | 要素を重ねて階層性と立体感をつくる |

> **サムネイルを描く習慣**: 数センチの小さな手描きラフで構図を素早く検証する。「小さく強い構図は大きくなっても強さを保つ」。

## 3. コントラストとジャンプ率

コントラストとは明暗比だけでなく、あらゆる「対比」を指す。種類は**大小・明暗・粗密・色・形**の5軸。

**ジャンプ率**（版面における文字サイズの変化率）:

| ジャンプ率 | 特徴 | 向いている用途 |
|-----------|------|--------------|
| **高い** | メリハリ・力強さ・熱量感 | 広告・ポスター |
| **低い** | 落ち着き・クール・アカデミック | 書籍・高品質ブランド |

> **原則**: 何かを強くするには、何かを弱くする必要がある。すべてを大きくしても何も目立たない。

## 4. マージンとホワイトスペース

| 用語 | 定義 |
|------|------|
| **マージン** | 要素間の空間・ページ天地左右の余白。版面構造を規定する |
| **ホワイトスペース** | 意図的に開放された「見せる余白」。孤立効果・空間性・洗練感 |
| **マイクロホワイトスペース** | 文字間・行間の小さな余白。積み重なって版面トーンを決定する |

**設計の原則**:
- 余白は「余ったスペース」ではなく、最初から設計する
- デッドスペース（役割のない中途半端な空間）は避ける

**マージン量とイメージの関係**:
- 広い → 上品・余裕感 / 狭い → 勢い・拡張感
- 天を大きく → 開放的 / 地を大きく → 上方へ浮かび上がる動き
- 不均一 → 個性的な印象

## 5. 視線誘導

| 役割 | 対象 | 設計の目的 |
|------|------|-----------|
| **時間軸のデザイン** | 広告・ポスター | 最初に見せる要素→後で読む要素の順序を設計 |
| **機能のデザイン** | 書籍・雑誌の段組み | 次にどこを読むか迷わない機能的誘導 |

**基本パターン**:
- 横書き: **Z型**（左→右→下の「乙型」）に視線が移動
- 縦書き: **N型**（上→下→左へ）
- 重要な要素は左上・上部に配置するのが自然な誘導

**アイキャッチャーの活用**: 大きな写真・高彩度の色・大きな文字で視線を捕捉し、そこから関連情報へ自然に誘導する。

> 視線検証の実践: ラフ段階で視線の流れを矢印で描き、意図した順序で情報が伝わるか確認する。

## 6. 揃えると崩す

**揃える（フォーマットデザイン）**: 要素の位置・サイズ・書体に統一ルールを設ける。見た目の美しさと「共通の土台で情報を見せる」機能的効果を両立する。

**崩す（意図的な逸脱）**: 揃えられたデザインの一部をあえて不揃いにし、印象を強める。

> **絶対原則: 「揃えてから崩す」**
> はじめから揃っていないと崩れた部分の魅力が伝わらない。対比（コントラスト）があってこそ逸脱が際立つ。

| 手法 | 効果 |
|------|------|
| **シンメトリー（対称）** | 安定感・誘目性。失敗しにくい。格調ある印象 |
| **アシンメトリー（非対称）** | 個性・緊張感・動き。シンメトリーのアクセントとして機能させる |

## 7. グリッドシステム

版面上に見えない格子状の枠を設け、要素をその格子の線に合わせてレイアウトする手法。モダンデザインの根幹を成す考え方。

**2つのメリット**:
- **統一性**: 共通グリッドによる一貫した整合感
- **自由度**: グリッドを柔軟に活用した動きのあるレイアウト

**グリッドの基本要素**:
- **段（カラム）**: 本文が入る縦の区画。段数が多いほど細かい情報整理が可能
- **溝（ガター）**: 段と段の間のスペース
- **マージン**: グリッド全体を囲む外側の余白

| 状況 | 推奨グリッド |
|------|-------------|
| 文字情報が多い（雑誌・書籍） | 3段以上のマルチカラム |
| ビジュアル中心（ポスター） | シンプルな1〜2分割 |
| 複雑な情報整理（カタログ） | ベースライングリッド併用 |

> 数値で揃えることと見た目で揃えることは別。錯視を考慮して目視調整を行うのがデザイナーの作業。

## 8. フォトディレクション基礎

### 写真の2つの性質

| 性質 | 説明 |
|------|------|
| **具象性** | 対象をそのまま見せる。イメージを限定・共有させる（報道性） |
| **抽象性** | 言葉を超えたイメージ・感情を喚起する（芸術性） |

### 光の制御

| 光の種類 | 特徴 |
|---------|------|
| **直接光（硬い光）** | 陰影がはっきり。コントラストが強い |
| **バウンス（反射光）** | 壁・レフ板に反射させた柔らかい光 |
| **ディフューズ（拡散光）** | トレーシングペーパー越しの柔らかい光 |
| **自然光（屋外）** | 天候に左右される。よい光を見つける作業 |

### 画角とレンズ選択

| レンズ | 焦点距離 | 効果 |
|--------|---------|------|
| **広角** | ～35mm | 遠近を強調。空間を広く見せる |
| **標準** | 50mm前後 | 人の目に近い。自然な画面 |
| **望遠** | 135mm～ | 遠くを引き寄せる。圧縮効果 |

**トリミング**: 作品性の高い写真はノートリミング（撮った状態そのまま）が原則。

### 被写界深度

| F値 | 被写界深度 | 効果 |
|-----|-----------|------|
| **小（開放）** | 浅い | ピント部分を強調。背景ボケ |
| **大（絞り込み）** | 深い | 広い範囲にピントが合う |

被写体と背景の距離差が大きいほど背景がボケやすい。望遠レンズほど被写界深度は浅くなる。
