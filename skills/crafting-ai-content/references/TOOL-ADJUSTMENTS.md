# ツール調整・最終仕上げ

AI生成画像を100点の完成品に仕上げるための、デザインツールを使った具体的な操作手順です。AIは60〜70点のたたき台を生成します。残り30〜40点の品質向上は、人間によるツール操作で実現します。

---

## 品質向上テクニック

### AI生成画像の「惜しい」を修正する

AIが生成した画像には、以下の典型的な問題が発生しがちです。デザインツールのAI機能を活用して効率よく修正しましょう。

#### 崩れた文字・不要なテキストの消去

AIは文字を「意味」ではなく「図形」として認識するため、意味不明な文字列や崩れた漢字が生成されることがあります。

**手順:**

1. AI生成画像をデザインツールにアップロードする
2. 画像を選択し、編集メニューから「AI消去」系の機能を選択する
3. 消したい文字の部分をブラシでなぞるように選択する（大まかな選択で十分）
4. 実行すると、AIが周囲の背景になじませながら選択部分を自然に消去する

**ポイント:**
- ブラシサイズは消去範囲に合わせて調整する（50%前後が扱いやすい）
- 複数回に分けて消去すると、より自然な仕上がりになる
- 消去後はクリーンなキャンバスとして、正確なテキストを改めて配置する

#### 細部の不自然さの修正

| 問題箇所 | 対処法 |
|---------|--------|
| 指・手のパーツが不自然 | AI消去ツールで該当部分を選択して削除し、差し替えるか隠すレイアウトに変更 |
| 物体の形状異常 | 異常部分をAI消去で除去し、自然な見た目に修正 |
| 影・光の矛盾 | 修正が困難な場合、再生成を検討（プロンプトに「光源の位置: 右上45度」等を明示） |
| 低解像度・ぼやけ | デザインツールの「高画質化」「アップスケール」機能を適用 |

## テキスト整形

### 正確なテキストをデザインツールで追加する

AI消去でクリーンになったキャンバスに、意図を込めた文字を配置します。

**手順:**

1. デザインツールのテキストツールを選択する
2. メインコピーやサブコピーを追加する
3. ブランドイメージや伝えたい雰囲気に合わせてフォントの種類・サイズ・色を調整する
4. デザイン4原則を意識して配置する

**デザイン4原則によるテキスト配置チェック:**

| 原則 | テキストへの適用 | チェックポイント |
|------|----------------|----------------|
| 近接 | 関連情報を一まとまりに | メインコピーとサブコピーは近接させる |
| 整列 | 見えない補助線で揃える | テキスト要素の左端・右端・中央を統一 |
| 反復 | 同じスタイルを繰り返す | 見出しには同一フォント・色を徹底 |
| コントラスト | 重要情報を大きく | メインコピーはサブコピーの2〜3倍のサイズ |

### フォント選択のルール（テキスト整形時）

- 1つのデザインで使うフォントは2種類まで
- 見出しと本文でサイズ・太さに明確なコントラスト（ジャンプ率）をつける
- 日本語フォントは豊富な種類から雰囲気に合わせて選択

## サイズ・アスペクト比変更

### AI画像拡張で用途に合わせたサイズに最適化する

AI生成画像が使用したい媒体のサイズ（アスペクト比）に合わない場合、デザインツールのAI画像拡張機能で元の世界観を保ったまま自然に背景を拡張できます。

**主要なアスペクト比と用途:**

| アスペクト比 | 用途 | 変換元として多い比率 |
|------------|------|------------------|
| 16:9 横長 | WebサイトヘッダーSNS横投稿 | 3:2 → 16:9 |
| 9:16 縦長 | InstagramリールTikTok | 1:1 → 9:16 |
| 1:1 正方形 | Instagram投稿Facebookタイムライン | 3:2 → 1:1 |
| 3:2 横長 | バナー広告・ブログアイキャッチ | 生成時の標準比率 |

**AI画像拡張の手順:**

1. 生成した画像をデザインツールにアップロードし、目標サイズのアートボードに配置する
2. 画像を選択し、「AI画像拡張」または「生成塗りつぶし」系の機能を選択する
3. 拡張したいサイズ（アスペクト比）と拡張方向を指定する
4. AIが元画像のスタイルを解析し、違和感のないよう背景を描き足す
5. 複数の拡張パターンが提案されるので、最も自然でイメージに近いものを選択する

### 拡張した「余白」を最強のメッセージスペースに変える

AI画像拡張の真価は単なるサイズ変更ではありません。新たに生まれた余白は、メッセージを配置するための絶好のスペースになります。

```
拡張後の余白活用例（横長画像の場合）:

[左側余白]         [中央: 元の画像]      [右側余白]
メインコピー    ←  AIが拡張した背景   →  サブコピー + CTA
```

**余白活用のポイント:**
- 左側余白: 視線が最初に向かう場所 → メインコピーを配置
- 右側余白: 視線の終点 → CTAや行動喚起を配置
- 上下の余白: ロゴや日時情報などの補助情報に活用

> 余白は情報を呼吸させるための重要な要素。AI画像拡張はこの重要な「余白」を意図的に創り出すテクニックでもある。

## 要素合成テクニック

### 複数AI案のコラージュ&リミックス術

AIは同じプロンプトでも毎回少しずつ異なる画像を生成します。「A案の人物は最高だけど、背景はB案の方がいい」という場合、複数の案の「いいとこ取り」をして一つの完璧なクリエイティブに仕上げる技法です。

#### ステップ1: 使いたい要素を切り抜く

1. 複数のAI生成画像バリエーション（A案・B案・C案）をデザインツールにアップロードする
2. 人物や特定オブジェクトを抽出したい画像を選択する
3. 「背景除去」機能を実行し、AIが瞬時に人物や主要オブジェクトのみを切り抜く
4. 不要なオブジェクトを除去したい画像は「AI消去」機能で該当部分を削除する

#### ステップ2: レイヤーを使って理想の構図を再構築する

```
組み立て順序（下から上のレイヤー構造）:

Layer 3（最前面）: テキスト・CTA要素
Layer 2: 切り抜いた人物・メインオブジェクト
Layer 1（最背面）: 背景画像
```

**具体的な手順:**

1. B案の「背景画像」をキャンバス全体に配置する（最背面レイヤー）
2. C案から切り抜いた「人物」を配置し、サイズ・位置を調整する（中間レイヤー）
3. タイポグラフィやテキスト要素を配置して全体のバランスを整える（最前面レイヤー）

**コラージュ時のチェックポイント:**
- 切り抜いた人物と背景の「光源の方向」が一致しているか
- 切り抜いたエッジが自然か（ぼかし処理が必要な場合あり）
- 全体の色調（カラートーン）が統一されているか

> AIの「偶然性」を人間の「意図」によって再編集しコントロールする高度なテクニック。AIが生み出した予期せぬパーツの組み合わせが、時に人間の想像を超える化学反応を起こすこともある。

## ハイブリッドワークフロー詳細

### AIにパーツ生成・人間が組み立てるアプローチ

AIにいきなり完璧な広告画像を生成させようとすると、意図しないレイアウトや情報優先順位の混乱が発生しがちです。代わりに、**AIには「パーツ（素材）」の生成に特化させ、最終的な「組み立て（レイアウト）」は人間が行う**ハイブリッドなアプローチが効果的です。

#### 各パーツのAI個別生成

**① アイキャッチ画像**

広告の中心となる、最も視覚的なインパクトが求められる画像。

```
プロンプト構成例:
[被写体の詳細説明]。背景はシンプルでクリーンな白。
スタジオライティング、高品質な写真

例: 未来的なガジェットを操作する、笑顔の若い女性。
    背景はシンプルでクリーンな白。スタジオライティング、高品質な写真
```

**② 背景画像**

広告全体の雰囲気を決定づける画像。アイキャッチを引き立てる、主張しすぎないものが適切。

```
プロンプト構成例:
[色・グラデーション]、[テクスチャ・効果の説明]。
[ムード: ミニマル/モダン/温かみ等]

例: 淡いブルーとパープルのグラデーション、
    抽象的で柔らかい光の粒子が舞う背景。ミニマル、未来的
```

**③ タイポグラフィ（文字デザイン）**

メインコピーを単なるテキストではなく、グラフィックとして生成させる手法。

```
プロンプト構成例:
「[テキスト内容]」という文字を、[スタイル説明]なタイポグラフィでデザイン。
背景は透明。

例: 「AI Design Revolution」という文字を、
    ネオンライトのように光る未来的なタイポグラフィでデザイン。背景は透明
```

#### デザインツールでの組み立て手順

```
ハイブリッドワークフロー全体像:

STEP 1: 各パーツをAIで個別生成
  ├── アイキャッチ画像（人物・商品・シーン）
  ├── 背景画像（グラデーション・テクスチャ）
  └── タイポグラフィ画像（文字デザイン素材）
         ↓
STEP 2: デザインツールでレイアウト組み立て
  1. 背景画像をキャンバス全体に配置（最背面）
  2. アイキャッチ画像の背景を除去して前面に配置
  3. タイポグラフィ画像を配置してメインコピーを際立たせる
  4. テキストツールでサブコピー・CTA等を追加
  5. フォント・サイズ・配色を調整して全体バランスを整える
         ↓
STEP 3: 品質チェックとエクスポート（→ SKILL.md Section 7参照）
```

**このアプローチの利点:**

| 観点 | 効果 |
|------|------|
| 品質コントロール | 各パーツを最適なプロンプトで生成できる |
| レイアウトの自由度 | 人間が意図的に構成を決定できる |
| 修正の容易さ | パーツ単位での差し替えが可能 |
| 一貫性の確保 | 背景・人物・テキストを個別管理 |

### テンプレート活用との組み合わせ

デザインツール内のプロ品質テンプレートとAI生成素材を組み合わせる方法も効率的です。

**手順:**
1. デザインツールのテンプレートから、作りたい広告イメージに近いものを選択する
2. テンプレートのレイアウト・配色・フォントの骨格はそのまま活用する
3. アイキャッチ写真のみAIで新規生成し、テンプレートの既存画像と差し替える
4. テンプレートのテキストを自分の広告内容に合わせて修正する

> ノンデザイナーにとって、ゼロからレイアウトを組むのはハードルが高い。テンプレート活用術なら、デザインの難しい部分はプロに任せ、最も伝えたい「メッセージ」と「ビジュアル」の生成に集中できる。
