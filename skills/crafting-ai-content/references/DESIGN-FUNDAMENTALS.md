## デザイン4原則の広告適用

デザインの普遍的な4原則は、AI生成広告の「プロンプト指示」と「出力評価」の両局面で活用できます。

### 近接（Proximity）

**原理**: 関連する情報は近づけてグループ化し、無関係な要素は離して配置する。

**なぜ機能するか**: 人間の脳は、物理的に近くにあるものを意味的に関連したものとして認識する（ゲシュタルト近接の法則）。広告では、この性質を利用して受け手が情報を素早く整理できるレイアウトを作る。

**広告クリエイティブへの適用**:

| 適用場面 | NGパターン | OKパターン |
|---------|-----------|-----------|
| 商品名と価格 | 画面の両端に分散配置 | 隣接させてひとまとまりに |
| メインコピーとサブコピー | 別々のブロックとして離す | 視覚的に一体のテキストブロックとして配置 |
| CTAボタンと関連情報 | ページ中程にCTA、関連説明が下部 | 関連説明の直下にCTAを配置 |
| ロゴとブランドスローガン | 別々の角に配置 | 近接させてブランドブロックとして統一 |

**プロンプト指示テンプレート**:
```
メインコピー「[コピー文]」とサブコピー「[サブコピー文]」を
一体のテキストブロックとして配置し、CTAボタンをその直下に置く
```

### 整列（Alignment）

**原理**: すべての要素を見えないガイドライン（左揃え・中央揃え・右揃え）に沿って配置し、視覚的な秩序を作る。

**なぜ機能するか**: 揃っていない要素の集合は、視線が定まらず認知負荷が増大する。整列により「見えない骨格」が生まれ、受け手は情報の構造を無意識に把握できる。

**広告クリエイティブへの適用**:

| レイアウトタイプ | 揃え方 | 向いている広告 |
|----------------|-------|--------------|
| テキスト主体バナー | 左揃え統一 | BtoB・情報訴求型 |
| ビジュアル主体バナー | 中央揃え | 感情訴求・ブランド広告 |
| 人物アイキャッチあり | 人物の視線方向にテキスト | 人物を活用した訴求全般 |
| シリーズ展開 | 全媒体で同じ基準線 | キャンペーン一貫性確保 |

**整列チェック（出力評価用）**:
- [ ] すべてのテキスト要素が同一の基準線上に揃っているか
- [ ] アイキャッチとテキストの境界が意図的に設計されているか
- [ ] ロゴ・CTAが揃えラインに対応しているか

**プロンプト指示テンプレート**:
```
全テキスト要素は左揃えで統一し、左端を基準ラインとする。
アイキャッチは右側1/3に配置してテキストと明確に分離する
```

### 反復（Repetition）

**原理**: 特定のフォント・色・スタイルを繰り返すことで、一貫性・まとまり・ブランド認知を強化する。

**なぜ機能するか**: 視覚的なパターンが繰り返されると、受け手は「同一のブランド・シリーズ」として認識する。反復は「ブランドの声」を視覚言語で表現する手段。

**広告クリエイティブへの適用**:

| 反復要素 | 効果 | 実装方法 |
|---------|------|---------|
| ブランドカラー | ブランド認知の強化 | 全プロンプトに同一カラーコードを記述 |
| フォントファミリー | プロフェッショナルな統一感 | 「見出しはBold・本文はRegular」を固定 |
| テキストブロックの位置 | シリーズ広告の一貫性 | 全テンプレートで「左上にロゴ」を固定 |
| アイコン・形状スタイル | デザインシステムの確立 | 「角丸12pxの矩形」を全ボタンに反復 |

**9バリエーション生成時の反復活用**:
```
# 共通設定（全9パターンで反復）
ブランドカラー: #[カラーコード]
フォントファミリー: [フォント名]
ロゴ配置: 左上

# 変更する要素（バリエーション部分）
メインコピー: AI生成（9パターン各自）
アイキャッチ: AI生成（9パターン各自）
```

### コントラスト（Contrast）

**原理**: 色・サイズ・形の「違い」を意図的に作ることで、情報の優先順位を示し視線を誘導する。

**なぜ機能するか**: 人間の視覚システムは「差異」に反応する。コントラストが高い要素は自動的に注目を集め、「これが最重要情報だ」というシグナルを無言で発する。

**広告クリエイティブへの適用**:

**ジャンプ率の設計**（見出しと本文のサイズ比）:

| ジャンプ率 | 印象 | 適した広告 |
|-----------|------|-----------|
| 高（3倍以上） | ダイナミック・インパクト | セール告知・緊急告知・イベント広告 |
| 中（1.5〜2倍） | バランス・信頼感 | 一般的な商品広告・BtoB |
| 低（1.2倍以下） | 上品・落ち着き | 高級品・文化的コンテンツ |

**コントラストの種類と広告への応用**:

| コントラストの種類 | 具体例 | 広告効果 |
|-----------------|-------|---------|
| 色彩コントラスト | 黒背景に白テキスト | 視認性の最大化 |
| サイズコントラスト | メインコピー72px・サブコピー24px | 情報優先順位の明確化 |
| 形状コントラスト | 直線的背景に丸いCTAボタン | ボタンの際立ち |
| 明暗コントラスト | 暗い背景に明るいアクセントカラー | 高級感と視線誘導 |

**プロンプト指示テンプレート**:
```
メインコピーを最大サイズ（他のテキストの3倍以上）で配置。
CTAボタンは背景と高コントラストな色（例：背景が暗ければ明るいオレンジ）を使用
```

## フォント選択ガイド（判断テーブル）

### フォント種別の印象と適用場面

| フォント種別 | 代表的なフォント | 与える印象 | 広告での適用場面 |
|------------|--------------|-----------|---------------|
| **明朝体（セリフ体）** | Noto Serif JP、游明朝 | 格調・伝統・品格・知性 | 高級品・和ブランド・信頼重視のBtoB |
| **ゴシック体（サンセリフ体）** | Noto Sans JP、ヒラギノ角ゴ | 現代的・クリア・親しみやすさ | Web広告・テック系・一般消費財 |
| **筆書体・手書き風** | 源暎こぶり明朝等 | 温もり・親密さ・手作り感 | 和食・伝統工芸・プレミアム体験 |
| **デザイン書体** | 用途に応じて選定 | 個性・インパクト・独自性 | イベント告知・特定テーマのキャンペーン |
| **欧文ボールド** | Montserrat ExtraBold等 | 力強さ・グローバル感 | スポーツ・テクノロジー・グローバルブランド |

### フォント選択の判断フロー

```
ターゲットは誰か？
├─ シニア・保守層 → 明朝体（読み慣れた形式）
├─ 若年層・トレンド層 → サンセリフ体（モダン）
└─ 特定趣味・コミュニティ → デザイン書体（世界観一致）

伝えたいトーンは？
├─ 信頼・誠実 → 明朝体 or ゴシック体（太め）
├─ 親しみやすさ → ゴシック体（細め〜標準）
├─ 高級感 → 明朝体 or 欧文セリフ体
└─ インパクト・緊急性 → 太いゴシック体 or 欧文ボールド

媒体は？
├─ 小サイズバナー（300×250等） → ゴシック体（視認性優先）
├─ ポスター・大型ビジュアル → 自由度高め
└─ SNS投稿 → ゴシック体（スマホ視認性優先）
```

### フォント使用のルール（広告特化）

**ルール1: 1デザインで2種類まで**
フォントを3種類以上混在させると、視覚的なノイズが増大し、ブランドの一貫性が失われる。

```
OK: 見出し → Montserrat ExtraBold
    本文  → Noto Sans JP Regular

NG: 見出し → Montserrat
    本文  → 游ゴシック
    CTA  → Arial
    ロゴ  → 別のフォント
```

**ルール2: サイズ・ウェイトでヒエラルキーを作る**

| テキスト要素 | サイズ比 | ウェイト |
|------------|---------|---------|
| メインコピー（最重要） | 基準（100%） | ExtraBold / 900 |
| サブコピー | 30〜50% | Medium / 500 |
| 本文・説明 | 20〜30% | Regular / 400 |
| 小さな注記 | 15〜20% | Light / 300 |
| CTA文字 | 40〜60% | Bold / 700 |

**ルール3: AIへのフォント指示は英語名で**
```
# 明確な指示例
font: "Noto Sans JP Bold, 72px" for headline
font: "Noto Sans JP Regular, 20px" for body text
```

## レイアウト・グリッドシステム

### グリッドシステムの基本

グリッドシステムは「見えない骨格」として、要素を規則正しく配置する枠組みです。

**カラムグリッドの種類と用途**:

| グリッドタイプ | 分割数 | 適した媒体 |
|-------------|-------|-----------|
| シングルカラム | 1 | 縦長スクロール広告・ストーリーズ |
| 2カラム | 2 | バナー広告（ビジュアル＋テキスト） |
| 3カラム | 3 | Webトップバナー・横長広告 |
| 4カラム | 4 | 雑誌広告・新聞広告 |
| 12カラム | 12 | 高度な調整が必要なケース |

**バナー広告のよくある2カラムレイアウト**:
```
┌─────────────────────────────┐
│  [アイキャッチ画像]  │ テキスト │
│       左50%       │  右50%  │
│                   │ ・コピー  │
│                   │ ・サブ   │
│                   │ [CTA]   │
└─────────────────────────────┘
```

### ジャンプ率（メリハリ）の設計

**ジャンプ率とは**: 見出し（最大テキスト）と本文（最小テキスト）のサイズ比率。

```
ジャンプ率 = 見出しサイズ ÷ 本文サイズ

例: 見出し72px ÷ 本文20px = 3.6倍（高ジャンプ率・インパクト重視）
    見出し36px ÷ 本文24px = 1.5倍（低ジャンプ率・落ち着き重視）
```

**広告目的別の推奨ジャンプ率**:

| 広告目的 | 推奨ジャンプ率 | 理由 |
|---------|-------------|------|
| セール・緊急告知 | 3倍以上 | 瞬時に「お得感」「緊急性」を伝える |
| ブランド認知 | 2〜2.5倍 | インパクトと品格を両立 |
| リード獲得（詳細情報あり） | 1.5〜2倍 | 読みやすさと情報量のバランス |
| 高級品・ラグジュアリー | 1.3〜1.8倍 | 上品な余白感と格調 |

### 情報のグルーピングと優先順位

良いレイアウトは情報を「意味のあるグループ」に分けて提示する。

**広告情報の3層構造**:
```
第1層（最重要・最大サイズ）:
  → メインコピー・キャッチコピー

第2層（補足・中サイズ）:
  → サブコピー・ベネフィット説明

第3層（行動喚起・目立つ色）:
  → CTAボタン・申し込みリンク

第4層（付加情報・最小）:
  → 注記・期限・ブランドロゴ
```

## 視線誘導パターン

### Z型レイアウト（バナー・LP向け）

人は横書きコンテンツを「Z」の形に視線を動かして読む傾向がある。

```
[スタート]                    [ここで情報を受け取る]
 ロゴ/ブランド   ────────▶   サービス名/商品名
         ╲
          ╲（視線が対角に移動）
           ╲
[補足情報]   ────────▶   [ゴール：CTA]
 サブコピー                  CTAボタン
```

**Z型レイアウトのプロンプト指示**:
```
Z型視線誘導を採用:
- 左上: ロゴ
- 右上: サービス名（最大テキスト）
- 左下: サブコピー・ベネフィット
- 右下: CTAボタン
```

**Z型が効果的な場面**:
- 横長バナー広告（728×90、970×250等）
- SNS投稿（横長・正方形）
- ランディングページのファーストビュー

### F型レイアウト（テキスト多め・情報量が多い広告）

人は情報量が多いコンテンツを「F」の形に視線を走らせる。

```
[全体を横に走る]
━━━━━━━━━━━━━━━━━━━━━━━━━━
▼
[少し下を横に走る]
━━━━━━━━━━━━━━━━━━━━━
▼
[左端を縦に走る]
┃
┃
┃
```

**F型が効果的な場面**:
- テキスト多めのWebバナー
- 複数商品・サービスの比較広告
- コンテンツ重視のSNS投稿

### 視線誘導の設計チェックリスト

- [ ] 「最初に何を見てほしいか」が明確に設計されているか
- [ ] 視線の流れが自然に次の重要情報へと誘導されているか
- [ ] CTAが視線の「終着点」に配置されているか
- [ ] 重要情報が視線の「盲点」（右下・端）に置かれていないか

## 色彩理論（三属性・心理・配色）

### 色の三属性

色を効果的に使うには、3つの属性を理解することが基礎となる。

| 属性 | 定義 | 広告での活用 |
|-----|------|------------|
| **色相（Hue）** | 赤・青・黄などの「色の種類」。色相環上の位置で表される | ブランドカラー・ターゲット訴求の核心 |
| **明度（Value）** | 色の「明るさ・暗さ」。白に近いほど高明度、黒に近いほど低明度 | 背景との視認性・コントラスト調整 |
| **彩度（Saturation）** | 色の「鮮やかさ・くすみ」。鮮やかなほど高彩度、グレーに近いほど低彩度 | 緊急性・落ち着き・高級感の演出 |

**三属性の組み合わせパターン**:

| パターン | 明度 | 彩度 | 印象 | 広告例 |
|---------|------|------|------|-------|
| パステル | 高 | 低 | 柔らかさ・女性的 | コスメ・育児・スイーツ |
| ビビッド | 中 | 高 | 元気・目立つ | セール・子ども・スポーツ |
| ダーク | 低 | 中〜高 | 高級感・シリアス | ラグジュアリー・BtoB |
| ニュートラル | 中〜高 | 低 | 上品・洗練 | プレミアム・ミニマル |

### 色の心理効果（広告文脈）

| 色 | 主な心理効果 | 広告での主な用途 | CTAへの適性 |
|----|-----------|---------------|------------|
| **赤** | 情熱・緊急・エネルギー・食欲 | セール・フード・緊急告知 | ◎ 高（緊急性を演出） |
| **青** | 信頼・誠実・知性・冷静 | 企業広告・金融・医療・テック | △ 中（信頼感があるが緊急性低） |
| **黄** | 明るさ・希望・注意・楽しさ | 子ども向け・注意喚起・ポジティブ | ○ 良（注目度高い） |
| **緑** | 自然・健康・安らぎ・成長 | 環境・ヘルスケア・食品（健康系） | △ 中 |
| **オレンジ** | 活気・親しみやすさ・食欲 | 食品・ポジティブキャンペーン・EC | ◎ 高（行動喚起に有効） |
| **紫** | 高貴・神秘・創造性・優雅 | 高級品・アート・占い・美容 | △ 中（用途依存） |
| **白** | 清潔・シンプル・開始 | ベースカラー・ミニマルデザイン | - ベース用途 |
| **黒** | 高級感・洗練・力強さ | ラグジュアリー・フォーマル | △ 高級感が強い場合 |

**AIへの色指定テンプレート**:
```
# 色は必ずカラーコードで指定する（認識精度向上のため）
primary-color: #0066CC（信頼感のある青）
accent-color: #FF6B35（行動喚起オレンジ）
background: #F8F9FA（軽いベースグレー）
```

### 配色テクニック

#### 3色ロール（ベース・メイン・アクセント）

広告配色の基本フレームワーク：

```
ベースカラー  70%  ── 背景・余白
              例: 白 / ライトグレー / ネイビー（暗い背景の場合）

メインカラー  25%  ── ブランドイメージを体現する色
              例: ブランドブルー / コーポレートレッド

アクセントカラー 5% ── CTAボタン・最重要テキスト
              例: メインカラーの補色 / 高彩度の色
```

**アクセントカラー選定のコツ**: メインカラーの補色（色相環で反対側）を使うと自然なコントラストが生まれる。

#### 色相関係と配色タイプ

| 配色タイプ | 色相環での関係 | 印象 | 適した広告 |
|-----------|-------------|------|-----------|
| **類似色配色** | 隣接する色（±30°） | まとまり・穏やかさ | ブランド広告・ライフスタイル系 |
| **対照色配色** | ある程度離れた色（60〜120°） | 変化・活動的 | キャンペーン・イベント告知 |
| **補色配色** | 反対側の色（±180°） | 最大コントラスト・インパクト | セール・緊急告知・注意喚起 |
| **同一色相配色** | 同一色のトーン違い | 洗練・統一感 | 高級ブランド・ミニマルデザイン |

#### トーン統一（明度・彩度を合わせる）

色相が異なっていても、明度と彩度（トーン）を揃えると統一感が生まれる。

```
例: パステルトーンで統一
  ピンク: #FFB3C6（高明度・低彩度）
  ブルー: #B3D9FF（高明度・低彩度）
  グリーン: #B3FFCC（高明度・低彩度）
  → 全体にやわらかく統一感のある印象

例: ビビッドトーンで統一
  赤: #FF1744（中明度・高彩度）
  青: #2979FF（中明度・高彩度）
  → エネルギッシュで統一感のある印象
```

## 余白・装飾の効果的使用

### 余白（ホワイトスペース）の役割

余白は「何もない空間」ではなく、情報を整理し視認性を高める積極的なデザイン要素。

**余白の4つの機能**:

| 機能 | 効果 | 実例 |
|-----|------|------|
| **要素の際立ち** | 周囲に余白があると各要素が独立して認識される | 白背景に配置したCTAボタン |
| **視線誘導** | 余白は視線を次の要素へ誘導する「道」になる | テキストブロック間の余白 |
| **可読性向上** | 行間・段落間の余白で文章が読みやすくなる | 長文説明が必要な広告 |
| **高級感の演出** | ゆったりとした余白は洗練された印象を作る | ラグジュアリーブランド広告 |

**AIへの余白指示テンプレート**:
```
# 情報が詰まりすぎている場合
「要素間の余白を広げて、各情報ブロックに十分な呼吸を与えてください。
 背景の見えるスペースを全体の40%以上確保してください」

# セクション間の余白を広げる場合
「各セクション間のスペースを現在の2倍に広げてください」
```

### 余白量の目安

| 余白タイプ | 小（詰め込み型） | 中（標準） | 大（余裕あり） |
|-----------|--------------|---------|-------------|
| 要素周囲 | 8px | 16px | 32px+ |
| テキストブロック間 | 12px | 24px | 48px+ |
| セクション間 | 24px | 48px | 80px+ |
| コンテナ内側 | 16px | 32px | 64px+ |

### 装飾の判断基準

装飾（罫線・アイコン・図形・テクスチャ）を加える際は、「なぜ必要か」を自問する。

**装飾の目的チェックリスト**:
- [ ] 情報を区切るため（罫線・スペーサー）
- [ ] 特定の要素を目立たせるため（囲み・背景色）
- [ ] ブランドの世界観を演出するため（テクスチャ・パターン）
- [ ] 視覚的な楽しさを加えるため（アイコン・イラスト要素）

**装飾の使用原則**:

| 原則 | 詳細 |
|-----|------|
| 目的なき装飾は除去 | 「なんとなくおしゃれだから」という装飾は情報を埋もれさせる |
| 装飾は情報を引き立てる脇役 | 装飾が主役になるとメッセージが伝わらなくなる |
| 統一感のある装飾スタイル | 角丸・直線など混在させない（反復の原則と連動） |
| 過剰装飾は安っぽさの原因 | 特に高級ブランド広告では装飾は最小限に |

### 一貫性とサプライズのバランス

デザインに「緩急」をつけることで、単調さを避けつつブランド認識を維持する。

| 要素 | 一貫させるもの | 変化させるもの |
|-----|-------------|-------------|
| **色** | ブランドカラー（反復） | アクセントカラーの使用量・位置 |
| **フォント** | フォントファミリー（反復） | サイズ・ウェイト（コントラスト） |
| **レイアウト** | グリッド・基準線（反復） | 写真とイラストの切り替え |
| **スタイル** | トーン&マナーの方向性（反復） | 時折のビジュアルサプライズ |

**サプライズの適切な使用場面**:
- キャンペーン第1弾：通常スタイル → 第2弾：同一テーマの意外なビジュアル
- SNSシリーズ投稿：同一テンプレート → 特別投稿で大きく異なるビジュアル
- A/Bテスト用：ベースデザインを保ちつつ一部要素を差し替え

## デザインの根幹：誰に・何を・どう

すべてのデザイン基礎は、コミュニケーションの3要素に集約される。

| 要素 | 問い | デザインへの影響 |
|-----|------|---------------|
| **誰に（ターゲット）** | 届けたい相手は誰か。何に悩み、何を求めているか | 色・フォント・スタイルの方向性 |
| **何を（メッセージ）** | 伝えたい最重要なことは1つに絞れているか | レイアウトの優先順位・ジャンプ率 |
| **どう（クリエイティブ）** | そのメッセージをどの「かたち」と「体験」で届けるか | 4原則・視線誘導・配色の統合的判断 |

**AIへの指示時の活用**:
```
# プロンプト冒頭に必ず含める
ターゲット: [具体的な属性・課題]
メッセージ: [1文で最重要な伝えたいこと]
クリエイティブの方向性: [トーン・スタイル・媒体]
```

AIは「どう」を強力にサポートするが、「誰に」「何を」の戦略判断は人間が担う。
これら3要素が明確なプロンプトほど、意図に近い出力が得られる。
