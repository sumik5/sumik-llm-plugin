# GRAPHICS-IMAGE-WORD-SYSTEM.md
# イメージワードによるデザインコミュニケーション

> デザインの印象を言語化し、クライアントとデザイナーの認識ズレをなくすための実践ガイド。
> 配色・書体・レイアウト・写真の4要素を印象軸ごとに整理する。
> 情緒性フレームワーク（6軸イコライザー）は `WEB-EMOTIONALITY.md` を参照。

---

## 1. イメージワードシステムの意義

「いい感じ」「女性向け」「高級っぽく」——クライアントの言葉はしばしば曖昧だ。
デザイナーはその言葉の背後にある「印象の意図」を正確に引き出す必要がある。

**イメージワードシステムの3つの機能:**

| 機能 | 内容 |
|------|------|
| **共通言語の確立** | 感覚的なイメージを言語化し、クライアントとデザイナーが同じ絵を見る状態をつくる |
| **要素への変換** | 抽象的な印象を配色・書体・レイアウト・写真の具体的な方針に落とし込む |
| **苦手ジャンルの攻略** | 得意でないデザインスタイルに取り組む際の指針として活用する |

## 2. 印象座標マップ（2軸フレームワーク）

デザインの印象は **「温度感」** と **「エネルギー量」** の2軸で位置づけられる。

```
                高エネルギー（躍動・訴求）
                        ↑
            [活気・明朗]    [力強・躍動]
    暖 ───────────────────────────── 冷
            [柔和・愛らしさ]  [格式・沈静]
                        ↓
                低エネルギー（静謐・品性）
```

| 象限 | 温度感 | エネルギー | 代表的印象キーワード |
|------|--------|-----------|-------------------|
| **活気・明朗** | 暖 | 高 | さわやか・元気・楽しい・ポップ |
| **力強・躍動** | 冷〜中 | 高 | インパクト・男性的・アーティスティック・斬新 |
| **柔和・愛らしさ** | 暖 | 低 | かわいい・やわらか・ナチュラル・清潔 |
| **格式・沈静** | 冷 | 低 | 信頼感・整然・高級感・スタイリッシュ・神秘的 |

## 3. 印象カテゴリ別デザイン要素指針

各カテゴリを **配色 / 書体 / レイアウト / 写真** の4要素で整理する。

### 3-A. 清潔・美的（Aesthetic / Clean）

澄んだ空気感と非日常的な美しさ。クリーンさを重視する製品・サービス・医療・コスメ系に適する。

| 要素 | 指針 |
|------|------|
| **配色** | 寒色・白を主役に。文字色はスミ（黒）でなくグレーへ落とす。彩度は全体的に低く抑える |
| **書体** | 癖のないスタンダード体（セリフ/サンセリフどちらも可）。装飾書体・太いウェイトは回避 |
| **レイアウト** | ホワイトスペースを大きく取る。情報量を削ぎ落とし、余白で透明感を演出する |
| **写真** | ガラス・鉱物・空・海・水面など透明感のある被写体。直線多め・シンプルな構図を優先 |

### 3-B. 活気・明朗（Vibrant / Bright）

前向きで希望に溢れたエネルギー。採用サイト・スタートアップ・子ども向け・スポーツ系。

| 要素 | 指針 |
|------|------|
| **配色** | 高彩度の暖色系をメインに。色数は3〜4色で賑やかさを出す。黄・オレンジ・黄緑が中心 |
| **書体** | 太いゴシック体・丸みのあるフォント。読みやすさ優先で装飾的すぎない書体を選ぶ |
| **レイアウト** | 要素を大きく配置しダイナミックに。斜めライン・オーバーラップも有効。余白は少なめ |
| **写真** | 笑顔・アクション・太陽光・カラフルな被写体。後ろ向きより正面・斜め向きの構図 |

### 3-C. 柔和・愛らしさ（Soft / Cute）

やさしさ・女性性・ロマンチックさ。コスメ・スイーツ・女性向けファッション・ウェディング系。

| 要素 | 指針 |
|------|------|
| **配色** | パステルトーン（淡いピンク・ラベンダー・ペールブルー）。グラデーションと相性が良い |
| **書体** | 丸みのあるゴシック・細い明朝体。筆記体・手書き風フォントも印象に合う |
| **レイアウト** | 中央揃え・対称構図が安定感を生む。装飾要素（リボン・ドット・花柄）を適度に加える |
| **写真** | 花・女性・空・水・雲など儚さや柔らかさを感じる被写体。ピント外しも幻想感に効果的 |

### 3-D. 自然・温かみ（Natural / Warm）

有機的なぬくもりと生命感。飲食・農業・オーガニック・ライフスタイル・クラフト系。

| 要素 | 指針 |
|------|------|
| **配色** | 土・緑・木目・アースカラーを基調に。彩度は中程度。白よりクリーム色の余白感 |
| **書体** | 手書き風・筆記体・明朝体。ゴシックを使うならウェイトを細くして柔らかさを出す |
| **レイアウト** | 余白をゆったり取り、グリッドに厳密に縛らない自由な配置。テクスチャ背景との相性良好 |
| **写真** | 自然光・素材感・手仕事・植物・食材。過度な加工を避けリアルな質感を活かす |

### 3-E. 格式・信頼（Formal / Trustworthy）

整然とした権威感と誠実さ。金融・法律・大企業コーポレート・公的機関系。

| 要素 | 指針 |
|------|------|
| **配色** | 紺・ダークグレー・黒を中心に。差し色は金や白に限定。彩度を低く保つ |
| **書体** | 明朝体（和文）・ローマン体（欧文）が格式を演出。ゴシックを使う場合は細いウェイト |
| **レイアウト** | グリッドに厳密に従い整然と配置。斜め配置・非対称は避ける。余白は中程度に均等配分 |
| **写真** | スーツ・建築・モノクロ変換も有効。人物は正面でなく斜め〜横向きの落ち着いた構図 |

### 3-F. 高貴・格調（Luxury / Elegant）

洗練されたスタイリッシュさと高級感。ハイブランド・ホテル・宝飾品・高級自動車系。

| 要素 | 指針 |
|------|------|
| **配色** | 黒・ゴールド・深みのある単色を基調に。色数は2色以内に絞る。余白を色として使う |
| **書体** | 細いセリフ体・細いサンセリフ体。大きいサイズで見せるタイポグラフィが有効 |
| **レイアウト** | 非対称・断ち落とし・大きなビジュアルと最小限のテキスト。余白を贅沢に使う |
| **写真** | 陰影・テクスチャ・光沢感を活かした撮影。被写体を中央に置かず大胆なトリミング |

### 3-G. 力強・躍動（Powerful / Dynamic）

圧倒的な存在感とエネルギー。スポーツ・格闘技・LP・キャンペーン系。

| 要素 | 指針 |
|------|------|
| **配色** | 高コントラスト。黒×赤・黒×白・ネイビー×オレンジなど対比の強い2色構成 |
| **書体** | 最重量ウェイト（Ultra Bold・Black）を大胆に。サイズのコントラストで視線を誘導 |
| **レイアウト** | 斜め配置・破格の大きさ・画面からはみ出す配置。動きと緊張感を意図的につくる |
| **写真** | アクション・動きのブレ・仰角・ローアングル。被写体が画面に迫ってくる構図 |

### 3-H. 神秘・暗黒（Mysterious / Dark）

不思議さ・近未来・サイバー感。テック系・音楽・ゲーム・ナイトクラブ系。

| 要素 | 指針 |
|------|------|
| **配色** | 黒・深紫・ダークネイビーをベースに、発光色（シアン・マゼンタ）をアクセントに少量添える |
| **書体** | 細い幾何学的サンセリフ体。和文は極細ゴシック。スペーシングを広めに取る |
| **レイアウト** | 情報を最小限に絞った余白多用か、デジタルノイズ的な情報密度のどちらか |
| **写真** | 夜景・光のエフェクト・影・逆光・レンズフレア。被写体の輪郭を溶かすような処理 |

## 4. クライアントヒアリングでのイメージ共有プロセス

曖昧なオーダーを具体的なデザイン指針に変換する3ステップ。

```
STEP 1: イメージワードの絞り込み
  └ 「きれい・かわいい・おしゃれ」の中でどれが一番近いですか？
  └ 同義語3〜4語を提示し、最も近い言葉を1つ選んでもらう

STEP 2: 印象座標での位置確認
  └ 「温かみを感じるもの」か「クールな印象」か？
  └ 「落ち着いた雰囲気」か「エネルギッシュな雰囲気」か？

STEP 3: NG印象の確認
  └ 避けたい印象・NG事例の共有（何でないかを決めると方向性が絞れる）
```

**ヒアリングのコツ:**
- デザイナーが印象軸を提案し、クライアントが「合っているか」を判断する形式にする
- クライアントに「どちらにしますか？」と委ねると迷いが生じる。選択肢は3つ以内に絞る

## 5. 苦手なデザインジャンルへの対処法

得意でないスタイルの依頼に直面した場合の手順。

1. **印象軸を特定する**: 依頼の言葉を2軸マップに当てはめ象限を確認する
2. **4要素を参照する**: Section 3 の該当カテゴリで配色・書体・レイアウト・写真の指針を確認する
3. **反義語から逆算する**: 「力強い」を理解するには「やわらか」との対比から入ると速い
4. **参照事例を収集する**: Pinterest・Behance でキーワード検索し視覚ライブラリを先に作る
5. **デザインの引き算を意識する**: 要素を足すより引くことで洗練度が上がることが多い

## 7. デザイン連想の技法

印象からデザイン要素を発想するための補助的アプローチ。

**連想展開の手順:**
1. イメージワードを受け取る（例: 「素朴な」）
2. 5感ベースで連想を広げる（触感・匂い・音・味・視覚）
3. 連想した実物の色・形・テクスチャをデザイン要素に変換する

```
例: 「素朴な」→ 手触り（ざらざら）→ テクスチャ背景 → 紙・布の素材感
例: 「クールな」→ 温度（冷たい）→ 金属・ガラス感 → 寒色+幾何学形状
例: 「高級感のある」→ 重さ（ずっしり）→ 黒・ゴールド・余白の贅沢な使い方
```
