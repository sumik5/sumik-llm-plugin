### P25: 万能告知プロンプト

**用途**: 新商品・新サービス・イベント・店舗オープンなど、何か新しいことを告知したい時に使える最も汎用的なテンプレート。

```
下記の要件で、[告知したい内容]のための広告画像を生成してください。

# 広告の目的:
- [例：新製品の認知度向上とWebサイトへの誘導]

# 主なターゲット層:
- [例：健康に関心のある30代から50代の男女]
  （AIに提案させる場合は「提案してください」）

# 伝えたい主要メッセージ（ベネフィット）:
- [例：手軽に始められて、毎日の健康習慣をサポートします]

# 必ず含めたい要素:
- メインコピー：[例：新しい健康習慣、はじめませんか？]
- 商品名/サービス名：[具体的な名称]
- CTA（行動喚起）：[例：詳しくはこちらをタップ]
- （任意）アイキャッチとなる画像の説明：[例：新鮮な野菜や果物と一緒に製品が置かれている、明るく清潔感のあるイメージ]

# 全体の雰囲気・印象:
- [例：信頼感がある、親しみやすい、健康的、ポジティブ]

# 画像の仕様:
- アスペクト比：[例：3:2 横長]
- その他：[例：文字は読みやすく、視認性の高いフォントでお願いします]
```

**活用ポイント**: 「伝えたい主要メッセージ（ベネフィット）」を明確にするのが最重要。機能の羅列ではなく「それがターゲットにとってどんな良いことをもたらすか」を具体的に示す。

### P26: 新商品・新サービス発表

**用途**: 新商品や新サービスのローンチ告知。「新しさ」と「期待感」を最大化する。

```
[商品名/サービス名] の「新登場」を告知する、魅力的な広告画像を生成してください。

# 広告の目的：
- 新商品/サービスの認知拡大と、先行予約/初回購入の促進

# 主なターゲット層：
- [例：新しいもの好きで、トレンドに敏感な20代の男女]

# アピールしたい商品の特徴・提供価値：
- [例：業界初の独自技術を搭載、従来品より30%性能向上、今までにない新しい体験を提供]

# 必ず含めたい要素:
- メインコピー: [例: ついに登場! 未来が、今あなたの手に。]
- 商品/サービス名: [具体的な名称]
- キャッチフレーズ（任意）: [例: 想像を超える、○○体験。]
- CTA: [例: 今すぐ予約して特典をゲット！]
- アイキャッチ: [新商品/サービスが最も魅力的に見えるクローズアップ画像、またはその使用シーンを想起させるイメージ]

# （任意）発売日/提供開始日

# 全体の雰囲気・印象:
- [例: 革新的、スタイリッシュ、期待感、高級感、限定感]

# 画像の仕様:
- アスペクト比: [例: 3:2 横長]
- デザイン指示: [例: 製品が光り輝いているようなエフェクト、背景に未来的なパターンを使用]
```

**活用ポイント**: 「何が新しいのか」「それによってどんな未来が手に入るのか」を明確に伝える。アイキャッチには製品そのものの美しさだけでなく、「使うことで得られる体験」を想起させるイメージを指示すると効果的。

### P27: スライド資料の表紙

**用途**: 企業紹介資料・プロジェクト企画書等のプレゼン用スライドの表紙。第一印象で信頼度を高める。

```
[会社名] 会社説明資料の表紙を画像生成してください。

# 広告の目的
- 会社案内資料の表紙として使用し、清潔感と先進性を強調することで企業イメージを向上させる

# 主なターゲット層
- [例：法人顧客（店舗運営者・小売業・飲食業の経営者）]
- [例：採用候補者（20代～40代のビジネスパーソン）]

# 伝えたい主要メッセージ（ベネフィット）
- [例：AIとデザインの力で、ビジネスを楽しく]
- 信頼感と親しみやすさを兼ね備えた企業姿勢

# 必ず含みたい要素
- メインコピー: [例：「Company Deck」（文字サイズ: 150px、行間: 180px）]
- サブコピー: [例：「[会社名] 株式会社 会社説明」（文字サイズ: 20px）]
- ロゴ: 左上に「[会社名]」ロゴ
- コピーライト: 右下に「Copyright © [会社名] Co., Ltd All Rights Reserved.」（文字サイズ: 12px）

# 写真要素（アイキャッチ）
- [例：右中央に正円の写真（PCでタイピングをしている手元）]

# 文字デザイン
- フォントはジオメトリックサンセリフ（Montserrat, Inter, Avenir Next 風）

# 全体の雰囲気・印象
- ミニマルでクリーン、先進的、信頼感がある
- 白を基調にアクセントカラー（水色・灰色）で構成
- 余白を大きく取り、要素の配置はバランスよく整列

# 画像の仕様
- アスペクト比: 3:2（横長、1800 × 1200px 推奨）
- 上下左右の余白: 120px以上
- その他: 超高解像度。文字はシャープで視認性が高く、配色は正確に再現。
```

**活用ポイント**: スライドの表紙は「論理」ではなく「感情」にアプローチするメディア。具体的な指示に加え「清潔感と先進性を強調することで企業イメージを向上させる」などの目的や与えたい印象も記載する。

### P28: 期間限定キャンペーン訴求

**用途**: セール・割引・プレゼントキャンペーンなど、期間限定施策の告知。「今すぐ行動しなければ損をするかもしれない」心理を刺激する。

```
[キャンペーン名] を告知し、参加/利用を促す広告画像を生成してください。

# 広告の目的：
- キャンペーン期間中の売上/申込数の最大化

# 主なターゲット層：
- [例：価格に敏感な既存顧客、新しい体験を求める潜在顧客]

# キャンペーンの主な内容・特典：
- [例：全品20% OFF、購入者全員に限定ノベルティプレゼント、先着100名様特別価格]

# 必ず含めたい要素：
- キャンペーンタイトル
- キャンペーン期間（例：〇月〇日まで！）
- メインコピー：[例：見逃し厳禁！年に一度のスペシャルオファー。]
- 特典内容を具体的に示す文言
- CTA: [例：今すぐチェックしてお得にゲット！、キャンペーンに参加する]
- アイキャッチ: [特典内容が魅力的に見える画像、楽しい雰囲気のイラスト、カウントダウンタイマーのような緊急性を示す要素]

# 全体の雰囲気・印象:
- [例：お得感、限定感、緊急性、楽しそう、見逃せない]

# 画像の仕様:
- アスペクト比: [例：3:2 横長]
```

**活用ポイント**: 人は「得をしたい」より「損をしたくない」（損失回避バイアス）に強く動かされる。「期間限定」「先着○○名」というキーワードで機会損失への恐れを活用する。アイキャッチには特典の魅力が一目で伝わる具体的なビジュアルを指示する。

### P29: ブランド認知向上

**用途**: 直接的な販売促進ではなく、ブランドの価値観・ストーリーを伝えて長期的なファンを育成する広告。

```
[ブランド名] のブランドイメージ向上を目的とした、心に残る広告画像を生成してください。

# 広告の目的:
- ブランドの認知度向上と好感度アップ、共感の醸成

# 主なターゲット層:
- [例：ブランドの価値観に共鳴する可能性のある潜在顧客層、ライフスタイルにこだわりを持つ層]

# 伝えたいブランドの核となる価値観・ストーリー:
- [例：環境への配慮と持続可能な社会への貢献、職人の手仕事による温もりと品質、挑戦し続けることの素晴らしさ]

# （任意）含めたい要素:
- ブランドロゴ（配置はAIに任せる、または具体的に指示）
- ブランドの世界観を象徴するようなビジュアル（人物、風景、抽象的なイメージなど）
- ストーリーを感じさせるコピー（AIに提案させるのも良い）

# 全体の雰囲気・印象:
- [例：信頼できる、共感できる、感動的、美しい、洗練されている、温かい、夢がある]

# 画像の仕様:
- アスペクト比：[例：3:2 横長]
```

**活用ポイント**: 機能的価値はコモディティ化しやすい。AIにはブランドが持つ「思想」や「物語」を伝える。ターゲットが共感し記憶に残るような「象徴的なビジュアル」と「心に響くコピー」の生成を目指す。

## ターゲット別

ターゲット層に合わせて表現方法を変えることで、広告効果が大きく異なる。

### P30: 若年層（10代〜20代前半）向け

**特徴**: 共感・トレンド・参加型が刺さる。SNSでの拡散も期待できる構成にする。

```
[商品名/サービス名/イベント名]を、10代～20代前半の若年層に向けてアピールする広告画像を生成してください。

# 広告の目的:
- [例：新商品の話題化、イベントへの参加意欲向上、ブランドへの親近感醸成]

# ターゲットのインサイト（推測）:
- [例：友達と共有したくなる、流行に乗り遅れたくない、自分らしさを表現したい、面白い体験をしたい]

# 訴求ポイント:
- [例：今SNSで話題の〇〇、友達と一緒に楽しめる、限定のレアアイテム、他とは違うユニークな体験]

# 必ず含みたい要素:
- メインコピー: [ターゲットが普段使う言葉遣いや共感できるフレーズ。例：「エモい体験、しよ？」「これ、マジ神！」]
- アイキャッチ: [ターゲットと同世代のモデルが楽しんでいる様子、トレンド感のあるイラストやグラフィック]
- ハッシュタグ提案: [例：#〇〇しか勝たん #チル体験]
- （任意）ユーザー参加型企画への誘導: [例：この投稿を引用してあなたの「推しポイント」を教えて！]

# 全体の雰囲気・印象：
- [例：ポップ、カラフル、楽しそう、おしゃれ、新しい、自分も参加したい]

# 画像の仕様：
- アスペクト比：[例：3:2 横長]
- デザイン指示：[例：動画映えするような動きのあるデザイン、手書き風フォントやスタンプ風のあしらい]
```

**活用ポイント**: 若年層にとってSNSはリアルなコミュニケーション空間の一部。AIに「若者言葉で」「ミーム風に」といった指示を出すことで、共感を呼ぶクリエイティブになる。「共有したくなる」仕掛けを盛り込む。

### P31: ビジネスパーソン（20代後半〜40代）向け

**特徴**: 課題解決・自己成長・論理性と信頼性が刺さる。ベネフィットを機能ではなく「変化」で語る。

```
[商品名/サービス名/セミナー名] を、20代後半から40代のビジネスパーソンに向けて訴求する広告画像を生成してください。

# 広告の目的:
- [例：業務効率化ツールの導入促進、ビジネススキルの学習意欲向上、キャリアアップ支援サービスの利用喚起]

# ターゲットのインサイト（推測）:
- [例：仕事の生産性を上げたい、専門知識を身につけて市場価値を高めたい、将来のキャリアに不安がある、信頼できる情報やサービスを探している]

# 訴求ポイント:
- [例：〇〇の課題を解決し時間を創出、専門家が教える実践的ノウハウ、あなたの市場価値を高める新しいスキル]

# 必ず含めたい要素:
- メインコピー：[ターゲットの課題に共感し解決策を提示する言葉。例：「その非効率、AIで終わらせませんか？」「未来を切り拓く、次世代型リーダーシップ講座」]
- 具体的なベネフィット：[例：導入後、残業時間30%削減！、明日から使える〇〇スキルが身につく]
- （任意）権威性・信頼性の担保：[例：導入企業〇〇社突破！、〇〇賞受賞、顧客満足度95%]
- CTA：[例：まずは無料資料請求、セミナーに申し込む、30日間無料トライアル]
- アイキャッチ：[スーツ姿で自信に満ちた表情のビジネスパーソン、課題解決の成果を示すグラフやデータ]

# 全体の雰囲気・印象:
- [例：プロフェッショナル、信頼できる、効率的、知的、課題解決志向、成長を促す]

# 画像の仕様:
- アスペクト比：[例：3:2 横長]
- デザイン指示：[例：青やグレーを基調とした落ち着いた配色、シャープでモダンなフォント]
```

**活用ポイント**: 「薬の成分（機能）」より「頭痛が治る（ベネフィット）」に人は反応する。機能ではなく「相手がどう変わるか」を語る。具体的なデータや実績をプロンプトに含めると説得力が増す。

### P32: 主婦・ファミリー層向け

**特徴**: 感情的な共感・安心感・お得感が刺さる。論理的説得より日常への寄り添いが効果的。

```
[商品名/サービス名/イベント名]を、主婦および小さな子どもを持つファミリー層に向けてアピールする広告画像を生成してください。

# 広告の目的：
- [例：時短家電の販売促進、子ども向け教育サービスの体験申し込み、家族向けイベントへの集客]

# ターゲットのインサイト（推測）：
- [例：毎日の家事をもっと楽にしたい、子どもの才能を伸ばしたい、家族みんなで楽しめる時間が欲しい、安全で信頼できるものを選びたい]

# 訴求ポイント：
- [例：たった5分で〇〇が完了！、子どもが夢中になる学び体験、家族の笑顔が増える週末、安心安全の国産素材]

# 必ず含めたい要素：
- メインコピー：[ターゲットの日常に寄り添い共感を呼ぶ言葉。例：「今日の献立、もう悩まない！」「家族の笑顔が、いちばんの宝物。」]
- 具体的なメリット：[例：忙しいパパ・ママの時間を創出、お子様の集中力アップ]
- （任意）安心・安全の訴求：[例：アレルギー対応、専門家監修、満足度No.1]
- CTA：[例：今すぐ試してみる、無料体験レッスンはこちら、お得なファミリーパックをチェック]
- アイキャッチ：[子どもと楽しそうに過ごす家族の温かい写真、時短調理で笑顔の様子、安心感を与える優しい色合いの製品画像]

# 全体の雰囲気・印象：
- [例：温かい、優しい、安心できる、楽しそう、お得感がある、実用的]

# 画像の仕様：
- アスペクト比：[例：3:2 横長]
- デザイン指示：[例：手書き風の温かみのあるフォント、パステルカラーを基調とした優しい配色]
```

**活用ポイント**: AIに「家族の笑顔が増える」といった具体的なシーンを指示することで、共感を呼ぶビジュアルが生まれる。「安心安全」は子ども向け商材では必須要素。

### P33: シニア層（60代以上）向け

**特徴**: 安心感・わかりやすさ・健康・生きがいに寄り添う。専門用語を避け、大きな文字・シンプルなデザインを基本とする。

```
[商品名/サービス名/イベント名]を、60代以上のシニア層に向けて訴求する広告画像を生成してください。

# 広告の目的:
- [例: 健康食品の販売促進、趣味の教室への参加者募集、シニア向けスマートフォンの利用促進]

# ターゲットのインサイト（推測）:
- [例: いつまでも健康でいたい、新しいことに挑戦してみたい、孫とコミュニケーションを取りたい、操作が簡単なものが良い、信頼できる情報が欲しい]

# 訴求ポイント:
- [例: ひざの痛みを気にせず毎日お散歩!、初心者でも安心・楽しい仲間ができる、スマホ操作もラクラク、専門家が丁寧にサポート]

# 必ず含めたい要素:
- メインコピー: [ターゲットの不安を解消し前向きな気持ちにさせる言葉。例: 「人生100年時代、まだまだ楽しむ!」「新しい趣味で、毎日が輝き出す。」]
- 具体的な便益: [例: 簡単な操作で、お孫さんとビデオ通話も。医師も推奨! 安心の健康成分。]
- （任意）安心感の醸成: [例: お電話一本で専門スタッフが対応、お客様満足度○○年連続No.1]
- CTA: [例: まずはお気軽にご相談ください、無料体験会に申し込む、資料請求はこちら]

# 全体の雰囲気・印象:
- [例: 安心できる、信頼できる、わかりやすい、健康的、穏やか、前向きになれる]

# 画像の仕様:
- アスペクト比: [例: 3:2 横長]
- デザイン指示: [例: 大きめで読みやすいフォントを使用、コントラストをはっきりさせた配色、情報を詰め込みすぎないシンプルなレイアウト、温かみのある暖色系の色使い]
```

**活用ポイント**: 「わかりやすさ」と「安心感」が最優先。「お電話一本で～」のような、デジタルに不慣れな方でも安心できるサポート体制の訴求が効果的。アイキャッチにはターゲットと同世代のモデルが生き生きと活動している姿を使う。

### P34: 特定の趣味・嗜好層向け

**用途**: アニメ・ゲーム・釣り・登山・特定音楽ジャンルなど、熱狂的なファンを持つニッチ分野向け。「専門性」と「仲間意識」で深く刺さる。

```
[商品名／サービス名／イベント名] を、[特定の趣味・嗜好] を持つ層に向けてアピールする、オリジナリティ重視の広告画像を生成してください。

# 広告の目的：
```

## プロンプト応用テクニック

### ネガティブプロンプトの活用

生成してほしくない要素を指定することで、意図した結果に近づける。

```
# 除外する要素（ネガティブプロンプト）:
- [特定の要素の排除例] 人物は入れないでください
- [品質向上例] 低画質、ピンぼけ、ノイズ、ウォーターマーク
- [雰囲気の回避例] 暗い、ホラー、不気味
- [特定の色の除外例] 赤、赤色
```

### プロンプト組み合わせアイデア

- **スタイルの混合**: 「浮世絵風」+「サイバーパンクなネオンライト」
- **感情の複合**: 「喜び」と「驚き」を同時に表現 → 「満面の笑みで目を見開いて空を指さす人物」
- **業種×表現スタイル**: 「飲食業界向けシズル感」+「レトロなアメリカンダイナー風スタイル」
- **長さの変化**: 短いキーワードで大きなイメージを生成後、徐々に詳細な指示を追加

### AI×AIでプロンプトを改善する方法

```
画像生成ツールで「[プロンプト]」を使いましたが、ありきたりな画像しか出てきません。
もっと独創的で、[望む印象]の画像を生成したいです。
現在のプロンプトをどのように改善すれば良いか、具体的なキーワードや表現のアイデアを5つ提案してください。
```

AIに生成・評価・プロンプト改善のサイクルを繰り返させることで、徐々にイメージに近い画像を生成できる洗練されたプロンプトへと近づく。

### 完成イメージを4要素に分解する思考法

1. **メインコピー**: 中心的なメッセージ
2. **サブコピー**: 補足情報
3. **アイキャッチ**: 視線を引きつけるビジュアル
4. **背景**: 全体の雰囲気

この4要素を明確にしてからAIに指示することで、より意図した結果を得やすくなる。配色はメイン・ベース・アクセントの3色で統一感を出す（配色比率の目安: ベース70%・メイン25%・アクセント5%）。
