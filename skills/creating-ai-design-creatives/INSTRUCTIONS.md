# AI広告クリエイティブ制作ガイド

## 概要

このスキルは、AI画像生成ツールを使って広告・マーケティングクリエイティブを制作するための実践ガイドです。

**対象媒体**: SNS投稿画像・バナー広告・ポスター・Webサイトビジュアル・サムネイル

**カバー範囲**:
- プロンプト設計（基本構造・YAML形式・Vibe Design）
- デザイン4原則の広告クリエイティブへの適用
- AI画像生成ワークフロー（手描きラフ活用・i2i・9バリエーション）
- よくある失敗パターンと修正方法
- 用途別プロンプトテンプレート（75パターン）の活用

**AIツールの役割**: AIは60〜70点のたたき台を瞬時に生成します。最終的な品質向上（残りの30〜40点）には、デザイン基礎知識と人間による調整が必要です。

---

## ユーザー確認の原則（AskUserQuestion）

**判断分岐がある場合、推測で進めず必ずAskUserQuestionツールでユーザーに確認する。**

確認すべき場面:
1. **プロンプト生成前**: 対象媒体（SNS投稿/バナー広告/ポスター/Web広告）
2. **プロンプト生成前**: ターゲット層（年代・性別・関心）
3. **デザイン調整時**: ブランドカラー/フォント指定の有無
4. **テンプレート選択時**: 業種と目的

確認不要な場面:
- デザイン4原則の適用方法（普遍的なベストプラクティス）
- 文字チェックの必要性（常に必須）
- 1クリエイティブ1メッセージの原則

## 1. プロンプト設計の基本構造

### 「誰に/何を/どう見せたいか」の3要素

広告クリエイティブのプロンプト作成では、コミュニケーションの根幹となる3要素を必ず含めます。

| 要素 | 内容 | 例 |
|------|------|-----|
| **誰に（ターゲット）** | 届けたい相手の属性・課題 | AI導入を検討している企業の経営者 |
| **何を（メッセージ）** | 伝えたい価値・ベネフィット | 明日から業務が変わる実践的なスキル |
| **どう（クリエイティブ）** | 表現方法・雰囲気・構成 | 未来志向・信頼感・Z型視線誘導 |

### 基本プロンプトテンプレート

```
下記の要件を満たす広告画像を生成してください。

# タイトル・企画
- [イベント名/商品名]

# サイズ
- アスペクト比: [例: 3:2 横長 / 1:1 正方形 / 9:16 縦長]

# 目的
- [バナークリック / イベント参加 / 商品購入 / ブランド認知向上]

# ターゲット
- [対象者の属性・課題・興味関心]

# 内容
## メインコピー
- [最も目立たせるキャッチコピー]

## サブコピー
- [メインコピーを補足する情報]

## CTA（行動喚起）
- [ユーザーに促したい行動]

## アイキャッチ
- [メインビジュアルの説明: 人物・シーン・スタイル]

## 背景画像
- [全体の雰囲気を決定づける背景]

# レイアウト
- [視線誘導の方向性: Z型/F型、要素の配置]

# 全体の印象・雰囲気
- [トーン&マナー: スタイリッシュ/親しみやすい/高級感 等]
```

### 詳細プロンプト（要素指定）

イメージが固まっている場合は、以下の要素を具体的に指定することで、より意図に近い出力が得られます。

| 指定項目 | 指定例 |
|---------|--------|
| フォント | `Montserrat ExtraBold, 72px` |
| 色（カラーコード） | `#FF4757（アクセントレッド）` |
| 配置（ピクセル） | `左から80px、上から200pxの位置` |
| 光の演出 | `ドラマチックな逆光（リムライト）` |
| 特殊効果 | `未来的な光の線、透明度60%` |

### YAMLフォーマットプロンプト

複数要素を構造的に指定したい場合はYAML形式が有効です。

```yaml
# 広告クリエイティブの詳細設計
canvas:
  size: 1200x800px
  background:
    type: gradient
    colors: ["#b74b8a", "#009688"]
    effect: "抽象的な光の波・光の粒子"

headline:
  text: "AI時代に必要とされる人材になるための戦略"
  font:
    family: "Noto Sans JP"
    size: 56px
    weight: 800

cta_button:
  text: "申し込みはこちら"
  background:
    gradient: {from: "#00c9a7", to: "#009688"}
  corner_radius: 12px

person:
  description: "20代男性、白Tシャツ＋ネイビージャケット、ノートPC操作、笑顔"
  position: right_center
  style: "リアル・自然光・スタジオライティング"

output:
  aspect_ratio: "3:2"
```

**メリット**: 細部まで精密にコントロールできる。テンプレート化することで部分変更が容易。
**デメリット**: 記述が複雑。カラーコード・フォント名・座標の知識が必要。

### Vibe Design（雰囲気を伝える手法）

細かく指定するよりも、全体の「雰囲気（Vibe）」を伝えることで、AIの創造性を最大限に引き出す手法です。初期のアイデア出しや、既存の枠にとらわれない独創的な表現に特に有効です。

```
- メインコピー: "Vibe Design しなよ"
- サブコピー: "今日から始める雰囲気デザイン"
- アイキャッチ: 20代の女性
- 雰囲気: スタイリッシュでファッションフォワードで、思わずWOWって感じの、マジカッケーデザイン
```

**Vibeを伝えるキーワード例**:
- 五感を刺激する言葉: 「キラキラ」「じゅわっと」「魂を揺さぶる」
- ジャンル参照: 「SF映画のワンシーンのように」「パンクロックのライブポスターのような勢いで」
- 感情をストレートに: 「とにかくハッピーな感じ！」「ちょっと切ないけど希望が見える」

> AIの「誤解」や「暴走」も新しいインスピレーションの源泉として楽しむ余裕が大切です。

## 2. デザイン4原則（広告クリエイティブ版）

広告のプロンプト指示にも、生成物の評価にも使える普遍的な4原則です。

### 近接（Proximity）: 関連情報をグループ化する

関連する要素は近づけ、無関係な要素は離す。脳は近くにあるものを関連づけて認識します。

**広告への適用**:
- 商品名と価格は近接して配置
- メインコピーとサブコピーは一まとまりに
- CTAボタンは関連情報の直下に

**プロンプト指示例**: `"メインコピーとサブコピーを一体のテキストブロックとして配置"`

### 整列（Alignment）: 見えない線で秩序をつくる

全ての要素を見えないガイドライン（左揃え/中央揃え/右揃え）に沿って配置します。

**広告への適用**:
- テキスト要素は一貫した左揃えか中央揃えで統一
- アイキャッチとテキストの基準線を揃える
- ロゴ・CTAの位置を整列基準に合わせる

**プロンプト指示例**: `"全テキスト要素は左揃えで統一、右側にアイキャッチを配置"`

### 反復（Repetition）: 同じルールを繰り返して一貫性をつくる

特定のフォント・色・スタイルを繰り返すことで、まとまりとブランドイメージを強化します。

**広告への適用**:
- シリーズ広告で同じフォントファミリーを使用
- 見出しには常に同じ色・スタイルを適用
- AIへの指示で同一スタイルを明示して複数生成

**プロンプト指示例**: `"このブランドカラー（#0066CC）を全シリーズで一貫して使用"`

### コントラスト（Contrast）: 違いを明確にして注目ポイントをつくる

色・サイズ・形の違いで情報の重要度を示し、視線を誘導します。

**広告への適用**:
- メインコピーはサブコピーの2〜3倍のサイズ
- CTAボタンには背景とのコントラスト強い色を使用
- 「ジャンプ率」（見出しと本文のサイズ差）を意識する

**プロンプト指示例**: `"メインコピーを他のテキストよりも3倍大きく、アクセントカラーを使用"`

### 視線誘導パターン

| パターン | 特徴 | 適した用途 |
|---------|------|-----------|
| **Z型** | 左上→右上→左下→右下の流れ | バナー広告・ランディングページ |
| **F型** | 上部横→少し下を横→左縦 | テキスト多め・情報量が多い広告 |

**Z型レイアウト指示例**: `"ロゴを左上、主要情報を中央、CTAを右下に配置。Z型の視線誘導を意識"`

→ 詳細（配色・フォント選択の判断基準・グリッドシステム）: `references/DESIGN-FUNDAMENTALS.md`

## 3. 色彩・フォント・レイアウト要約

### 色彩心理の広告活用

| 色 | 与える印象 | 広告での活用場面 |
|----|-----------|---------------|
| 赤 | 情熱・緊急・エネルギー | セール・緊急性を訴求するCTA |
| 青 | 信頼・誠実・知性 | 企業広告・B2B・金融 |
| 黄 | 明るさ・希望・注意 | 子ども向け・注意喚起 |
| 緑 | 自然・健康・安らぎ | 環境・ヘルスケア関連 |
| オレンジ | 活気・親しみやすさ | 食品・ポジティブなキャンペーン |
| 紫 | 高貴・神秘・創造性 | 高級品・アーティスティック |
| 白 | 清潔・シンプル | ベースカラー・ミニマルデザイン |
| 黒 | 高級感・洗練・力強さ | ラグジュアリー・フォーマル |

**配色バランスの基本**:
- ベースカラー（70%）: 背景など広い面積を占める落ち着いた色
- メインカラー（25%）: ブランドイメージを象徴する色
- アクセントカラー（5%）: CTA・重要情報を際立たせる色

### フォント選択の判断基準

| フォント種別 | 与える印象 | 適した場面 |
|------------|-----------|----------|
| 明朝体（セリフ体） | 格調・伝統・品格 | 高級品・和風・読み物コンテンツ |
| ゴシック体（サンセリフ体） | 現代的・クリア・汎用性 | Web広告・テック系・一般的な広告 |
| 筆書体・手書き風 | 温もり・親密さ・和の雰囲気 | 和食・手作り感を出したいブランド |
| デザイン書体 | 個性・インパクト | イベント告知・特定のテーマ表現 |

**フォント使用のルール**:
- 1つのデザインで使うフォントは2種類まで（混乱防止）
- 見出しと本文でサイズ・太さに明確なコントラスト（ジャンプ率）をつける
- 長い本文には可読性を優先したゴシック体

→ 詳細（グリッドシステム・余白活用・視線誘導の詳細手法）: `references/DESIGN-FUNDAMENTALS.md`

## 4. AI画像生成ワークフロー

### 手描きラフ → AI変換フロー

専門知識がなくても、手描きのラフスケッチをAIに読み込ませることで、自分のイメージをAIに正確に伝えられます。

**ステップ**:
1. **ラフを描く**: どこに何を置きたいかを示す。棒人間・丸・四角・英文字で位置指定するだけでOK
2. **AIに入力**: 画像をアップロードするか、言葉で位置を説明する
3. **プロンプトで補足**: 色味・雰囲気・テキスト内容・アイキャッチの詳細を追加指定する

```
この手書きラフ（添付画像）のレイアウトを再現し、下記の要件で広告画像を生成してください。

# メインコピー: [キャッチコピー]
# アイキャッチ: [人物・シーンの詳細説明]
# 全体の印象: [信頼感・革新的・親しみやすい 等]
```

### i2i（Image-to-Image）活用

既存の画像を元に新しいバリエーションを生成する手法です。

| 活用パターン | 内容 |
|------------|------|
| **スタイル変換** | 製品写真を水墨画風・イラスト風などに変換 |
| **要素の追加・変更** | 背景を別のシーンに差し替え、不要な人物を削除 |
| **モックアップ作成** | ロゴやキャラクターを多様なポーズ・場面で展開 |
| **画像の拡張（アウトペインティング）** | アスペクト比が合わない画像を自然な形で拡張 |

**著作権の注意**: 他者が作成した画像・許諾を得ていない人物写真の無断使用・改変は法的問題になる可能性があります。

### 9バリエーション生成手法

1つのプロンプトで複数の方向性を同時に探索する手法です。

```
下記要件で広告画像を9パターン生成してください。

# 共通設定
- タイトル: [サービス名]
- CTA: 今すぐ詳細をチェック！

# バリエーション指示
## メインコピー
- （9パターンそれぞれターゲットに合わせてAIが提案）

## アイキャッチ
- （9パターンそれぞれコピーに合わせたビジュアルをAIが提案）

## レイアウト
- 9つはそれぞれ異なる代表的な広告レイアウト構造を採用

## 印象
- 9パターンは全く違うテイストで
  （例: ポップ・知的・大胆・ミニマル・ドラマチック…）
```

**活用場面**: A/Bテストの初期仮説立案・複数のアイデアを短時間で比較検討

### ハイブリッドワークフロー（AI + 手動仕上げ）

AIが生成した画像はたたき台（60〜70点）として扱い、デザインツールで最終仕上げします。

```
1. AI生成（たたき台作成）
   ↓
2. 崩れた文字・不自然な要素を削除（デザインツールのAI消去機能）
   ↓
3. 正確なテキストをデザインツールで追加
   ↓
4. フォント・サイズ・配色を調整（デザイン4原則に沿って）
   ↓
5. サイズ最適化（使用媒体に合わせてアスペクト比を調整・拡張）
   ↓
6. 高画質化処理
```

→ ツール操作の詳細: `references/TOOL-ADJUSTMENTS.md`

## 5. 頻出5課題と解決策

AIが生成した画像によく見られる問題と、対処法をまとめます。

### 課題①: 文字の破綻・読みにくさ

| 症状 | 原因 | 解決策 |
|-----|------|--------|
| 意味不明な文字列・崩れた漢字 | AIが文字を「意味」ではなく「図形」として認識 | デザインツールで文字を消去し、改めて追加 |
| 不自然なカーニング・行間 | 文字の「形」として学習しているため間隔が不安定 | デザインツールで手動調整 |
| フォントのバランスが悪い | 特定フォントの特徴を正確に再現できない | 画像は背景のみ生成、文字はデザインツールで追加 |

**チェックポイント**: 「読めるか？」→「意味が通じるか？」→「ブランドイメージと合っているか？」の順に確認

### 課題②: 細部の不自然さ・物理法則の無視

| 症状 | 対処法 |
|-----|--------|
| 指の数がおかしい・関節が不自然 | 再生成を試みるか、人物の手を隠すレイアウトに変更 |
| 物体の形状異常（カップの持ち手が2つ等） | 該当部分をAI消去ツールで削除し差し替え |
| 影と光の矛盾 | 「光源の位置を明確に指定」するプロンプトで再生成 |

**予防策**: プロンプトに「8K品質、フォトリアリスティック」を明示することで改善する場合あり

### 課題③: ブランドイメージとのズレ

| 症状 | 解決策 |
|-----|--------|
| 指定色と微妙に違う | カラーコード（`#RRGGBB`）で指定、不一致時はデザインツールで調整 |
| 「高級感」を求めてもチープに見える | 具体的なブランドの参照画像を添付してi2iで再生成 |
| ターゲット層との乖離 | ターゲットの属性をより具体的に記述（年代・職業・ライフスタイル） |

**根本原因**: AIは一般的な「美しさ」の概念は理解できるが、特定ブランドの独自文脈・歴史・価値観の再現は困難

### 課題④: 情報の過不足・優先順位の混乱

| 症状 | 解決策 |
|-----|--------|
| 最重要メッセージが埋もれる | プロンプトで「メインコピーを最大サイズで、他の3倍以上大きく」と明示 |
| CTAが目立たない | CTAの位置・色・サイズを具体的に指定、背景とのコントラストを強調 |
| 情報過多で混乱 | 「1クリエイティブ1メッセージ」の原則を徹底 |

**鉄則**: 1つの広告で伝えるメッセージは1つだけ。優先順位を明確にしてプロンプトに反映

### 課題⑤: 複数画像生成時の一貫性欠如

| 症状 | 解決策 |
|-----|--------|
| キャラクターの外見が変わる | 参照画像機能（i2i）でキャラクターを固定して生成 |
| シリーズで色味がブレる | 具体的なカラーコードを全プロンプトに共通記述 |
| スタイルが安定しない | 同じベースプロンプトから値のみ変更する「テンプレート方式」を採用 |

**注意**: AIの生成プロセスには意図的なランダム性があります。一貫性が求められる場合は最終調整を人間が担う前提でワークフローを設計する

## 6. プロンプトテンプレート（サマリー）

> **テンプレート使用前に AskUserQuestion で確認してください**:
> - 「業種/目的を教えてください（例: 飲食店/新メニュー告知、BtoB SaaS/セミナー告知）」

### カテゴリ別テンプレート概要（全75パターン）

| カテゴリ | 件数 | 用途 |
|---------|------|------|
| **汎用テンプレート** | 5件 | 万能告知・新商品発表・スライド表紙・キャンペーン・ブランド認知 |
| **ターゲット別** | 5件 | 学生・シニア・ビジネスパーソン・主婦・経営者 向け |
| **目的・課題別** | 7件 | 採用・教育・イベント・EC・アプリ・資金調達・社会貢献 |
| **業種別** | 11件 | 飲食・美容・医療・不動産・金融・旅行・スポーツ・音楽・教育・IT・ファッション |
| **クリエイティブ表現** | 8件 | SNSサムネイル・動画サムネ・インフォグラフィック・ミームデザイン等 |
| **上級テクニック** | 5件 | シリーズ展開・A/Bテスト・リターゲティング・インタラクティブ広告 |

### 各カテゴリの代表例

**汎用告知（万能）**:
```
下記の要件で、[告知したい内容]のための広告画像を生成してください。

# 広告の目的: [例: 新製品の認知度向上とWebサイトへの誘導]
# 主なターゲット層: [例: 健康に関心のある30代〜50代の男女]
# 伝えたい主要メッセージ（ベネフィット）: [例: 手軽に始められて毎日の健康習慣をサポート]
# 必ず含めたい要素:
  - メインコピー: [例: 新しい健康習慣、はじめませんか？]
```

## 7. 最終調整・品質チェック

### 品質チェックリスト

AI生成後、以下の順番でチェックします。

#### 文字チェック（最優先）
- [ ] 文字は正しく読めるか（意味不明な文字列・崩れた漢字がないか）
- [ ] メインコピーの内容が意図通りか
- [ ] CTAテキストが正確か

#### 細部チェック
- [ ] 人物の指・手・顔のパーツが自然か
- [ ] 物体の形状・光・影に不自然な点がないか
- [ ] 解像度は十分か（低解像度は高画質化処理を適用）

#### デザイン原則チェック
- [ ] デザイン4原則（近接・整列・反復・コントラスト）が守られているか
- [ ] 視線誘導はZ型またはF型で自然か
- [ ] 情報の優先順位（メインコピー > サブコピー > CTA）が明確か

#### ブランド整合性チェック
- [ ] ブランドカラーが正確に反映されているか
- [ ] フォント・雰囲気がブランドイメージと合っているか
- [ ] ターゲット層に好感を持たれる内容か

#### 著作権チェック
- [ ] 使用した画像素材の著作権・利用規約を確認済みか
- [ ] AI生成画像の商用利用が許可されているツールを使用しているか
