## 1. 文章は「人を動かす」ために書く

### 核思想

文章は何のために書くのか。答えは一つ——**人を動かすために書く**。

記録でも報告でもなく、読んだ相手の行動・判断・感情を変える力を持つ言葉。その力は、書き言葉の習練によって獲得される。

> 「ポイントは、文が『上手いか、下手か』にあるのではなく、『人を動かすか、否か』にある」（齋藤孝）

### 実用的文章力の本質

「実用的」とは「人を動かす」という意味に等しい。一見拙くても心を動かす文章は力を持つ。野口英世の母シカの手紙がその例だ。文章力の訓練は技巧ではなく、**発見力・認識力・段取り力**の訓練である。

### 「意味をつかむ握力」を持つ

書ける人と書けない人を分けるのは文章力ではなく、「意味をしっかりつかみ取る握力」の有無だ。段取りを組む力があれば、大半の文書は書ける。

## 2. エッセイの本質と「書く生活」

### エッセイとは何か

| 文章形式 | 定義 |
|---------|------|
| 作文 | 感想や出来事のプロセスを書く |
| 読書感想文 | 本についての感想を書く |
| **エッセイ** | テーマを決め、語るうちに本や経験が引用される。知的な「捉え直し」と「発見」が必要 |

エッセイの構造：**テーマ → 捉え直し → 発見**。

あらすじを書いて終わる読書感想文はエッセイではない。書き手の「新しい認識」を提示できてはじめてエッセイと呼べる。

### 「書く生活」と「書かない生活」の差

毎週エッセイを書くことを課した学生たちに起きた変化：

- 書くことを意識しながら一週間を過ごすようになった
- 「これは発見にならないか」という意識でものを見るようになった
- 認識力・発見力が向上した

**「書く生活」を始めると、暮らし方・ものの見方が変わる。** これはおしゃべりでは身につかない変化だ。

### おしゃべりと書くことの決定的な違い

おしゃべりには「認識力」という特殊な力は必要ない。しかし書くという作業は違う。認識的な発見をうながす、いい意味での圧力が書く作業には不可欠だ。

「目のつけどころがいい」人物は書くことで鍛えられる。おしゃべりだけでは手に入らない力だ。

### エッセイを企画書のように書く

エッセイと企画書は実はつながっている。どちらも：
1. 発見・新しい認識をはっきりと打ち出す
2. タイトルで読者を引き込む
3. 詳細を展開し、最後に結論を提示する

「論語カルタを授業に使ってみたら」——こんなタイトルでエッセイが書けると、そのままビジネス企画書になる。日常的にエッセイを書くことで、ユニークな視点が自在に操れるようになる。

### 「凡庸さを打ち破るコツ

「チームワークが大切だと思いました」——これは最悪の結論だ。凡庸すぎる。

打ち破る方法：
1. **別のキーワードで言い換える**。チームワークの状態をイメージし、別の言葉が浮かばないか考える。「あやとり感覚」のような独自表現が生まれれば勝ち。
2. **異質なもの同士をつなぐ**。「通勤地獄の解消にあやとり感覚」——タイトルにするだけで読者は引き込まれる。
3. **一般論を乗り越えてから書き始める**。「一般的にはこうだが、自分はこう見る」という切り口を持つ。

## 3. 学生のための文章術

### 読書感想文 → 読書エッセイへの転換

**やってはいけないこと**：
- 「○○を読んで」というタイトル（本が主役になってしまう）
- あらすじの紹介で終わる
- 著者にひれ伏す姿勢

**正しいアプローチ**：
- **書き手が主役**。本はあくまで素材だ。
- 「読書エッセイ」と読み替える。
- 「この私をここまでその気にさせてくれた。なかなか大したものだ」——これくらい「上から目線」でいい。
- 「生意気さ」は好きだという前提があって初めて光る。

**視点の見つけ方（3段階）**：
1. 「肩入れしたい登場人物」を挙げる（主人公以外でいい。マイナーキャラでも）
2. その人物の視点で物語を捉え直す（スピンオフ的アナザーストーリー）
3. 登場人物を弁護士になったつもりで論理的に弁護する

**引用のコツ**：一冊から気の利いたセリフを一つ拾えれば上出来。そのセリフを核に文章を書く。

### 小論文で「光る」ための3戦略

採点者は上から2割の「光っているな」と感じる小論文を探している。8割は凡庸だ。

**戦略①：ビッグネームを放り込む**

ニーチェ、フーコー、ドラッカー、ブルデュー……難解な思想家の「定義だけ」「いいことだけ」を拾い読みし、「これ、来るかも」と感じた概念を記憶しておく。

試験会場で無理やりでもつなぎ合わせて盛り込む。「よくここにはめ込んだ」と評価される強引さが、むしろ加点される。

**戦略②：課題文の論理構造を他の事象に適用する**

「自由という名の不自由」という概念を抽出し、他の現象で切ってみる。課題文の分析で終わらず、「この視点で見ると他のことも……」と視野を広げると採点者は「違う次元の小論文だ」と感じる。

**戦略③：ディベート的発想**

「賛成か反対か」の二項対立に縛られない。「ここは妥当だが、この部分は行き過ぎている。修正するとこうなる」という、課題文をブラッシュアップする論法を取る。

### 自己アピール文（「書く就活」）

**核心**：自分の人生の転換点・分岐点を示す「断面図」を見せる。ふわふわした全体像ではなく、一つのクライシス（危機）とその克服を切り口にする。

| 要素 | 内容 |
|-----|------|
| **失敗をアピール** | 失敗 + 原因の冷静な分析 + 修正能力の実績。企業は「修正能力がある人材」を求めている |
| **転換点を盛り込む** | アイデンティティを失いかけた瞬間 → 回復のプロセス → 現在の自分 |
| **名づけをする** | 「失敗から得たプロ意識」のように、自分の変化に名前をつける。採用者に「ここまで成長している」と伝わる |
| **パッションを示す** | その会社で何がしたいか、どう変えたいか、社会においてどんな意味を持つか。「就職させてください」ではなく「この会社で実現したいことがある」という文章 |

**NG例**：「自分はコミュニケーション能力が高いと思います。御社で活かしたいです」
→ 具体性ゼロ。面接官が掘り下げる余地がない。

**OK例**：失敗エピソード + 自己分析 + 修正のプロセス + アイデンティティへの名づけ
→ 面接官が「そのとき、どうした？」と深掘りできる入り口になる。

## 4. メールの「身体性」と「プラスα」

### メールに「身体性」を持ち込む

通常、メールには身体性がない。声のトーンも表情も伝わらない。だからこそ、意図的に身体性を入れると際立つ。

**「！」マークの効果**：

上司から無茶ぶりのメールが来たとき——「よろこんで！」と返信する。本音は「えー、マジで？」でも、「！」をつけるだけで印象が変わる。

上司も「仕事が山積みなくせに『よろこんで！』はないよな」と思いつつ、「そういうメールを送ってくるところが可愛い」と感じる。その姿勢が積極的に評価される。

**身体性の出し方**：
- 緩い文体にする
- ユーモアや雑談を入れる
- やる気を文体で表現する

### プラスαのメール

実務的な内容に「少し個人的なコメント」を加える技術。

```
例）
実務内容のやりとり
＋
「最近読んだ本で、あなたの仕事に関係しそうな記述がありました。
 『○○』という本の○○章に……（簡単な紹介）。
 特にお返事は結構ですので」
```

このような「お中元・お歳暮感覚メール」を年1〜2回出すだけで、滅多に会えない相手との「つながってる感」が生まれる。コストゼロ、送られた側は嫌な気持ちにならない。

**原則**：用件だけじゃ「ない」ところをメールで出す。あなたのパーソナリティが伝わり、「A社への発注はあの人に頼もう」と思わせる存在になる。

### 「手紙感覚メール」の効果

映画の感想や本の批評など、まとまった文章を「ガン！」と送ることで人間関係が深まる。

人間は話す行為よりも書く行為のほうが、自分の人間性・人間の深さをさらけ出す。相手の心の中に入っていける可能性が高い。「あの人、こんなとこまで考えてたのか」という発見を与えられる。

### 打診を断るときの注意

メールは「薄氷」だ。

軽い気持ちで断ったつもりが、相手にとって大事な打診だった——そういうことが起きやすい。断った後に仕事が一切来なくなることもある。

**断り方の注意点**：
- 予算の話だけで断ると「カネの亡者」と見られかねない
- クールな事務処理文体は感情を削ぎ落とす（「申し訳ありません。できません」では身体性がなさすぎる）
- 日頃の「雑談メール」の蓄積が断り時の緩衝材になる

電話と組み合わせる方法：メールで「どうかな？」と打診し、電話でニュアンスを確認してから詰める。強引さを感じさせずスムーズに進む。

## 5. 評価されるワンランク上の文章力

### 一人弁証法

弁証法とは「正・反・合」の論理展開。これを一人の文章の中で展開する技術が「一人弁証法」だ。

```
正（賛成意見）
  ↓
反（反対意見・クレーム）
  ↓
合（両者を踏まえた結論・提案）
```

**効果**：
- 書き手の柔軟性が伝わる
- 「この人は複数の立場から物事を考えられる」という印象を与える
- 文章の推進力が増す（同じ方向の一点突破より、対立を乗り越えた結論のほうが読者を深く納得させる）

**実践方法**：

「クレーマー」として自分の意見に上手にクレームをつける。次に「弁護人」として反論する。この往復が文章の動きを作る。政治学者・丸山真男が愛用した「問答形式」はこの応用だ。

**注意**：文脈をつなげる力がないと、単に議論が混乱しているだけに見える。段取り力が前提。

### 視点移動能力

複数の視点を持って同一対象を見る能力。これが企業でも評価される。

| 視点 | 内容 |
|-----|------|
| **お客さん視点** | 作る・売る側ではなく、買う・使う側から自社を眺める |
| **クレーマー視点** | 自分の意見に上手にクレームをつけて鍛える |
| **弁護人視点** | クレームへの反論を弁護士のように組み立てる |

セブン-イレブン会長・鈴木敏文の言葉：「皆さんはこれまで当社のお客さんだったんだから、そのお客さんの視点を絶対に忘れないように」

「お客さんの立場から自分の会社を本気でながめ直す能力」を持った人間は組織で重宝される。

### 「凡庸さは恥」——独自視点の鍛え方

「結局のところ、人生で大切なのは人間性を磨くことだ」——これをテレビで言うだけならまだしも、活字にすると意味の含有率がほぼゼロになる。

**凡庸さを避ける原則**：
1. 一般論を「乗り越えてから」書き始める。「一般的にはこうだが、自分はこう見る」という切り口を持つ。
2. 奇をてらうことが目的ではない。今まで誰も注目しなかった部分に光を当て、別の着眼点から本質を押さえる。

フランス人は小論文・エッセイを書く時、「他人と同じことを言ったら恥」という強迫観念がある。「他人とは違う角度でものを見る」練習を幼少から積んでいる。

### 「マイナー感覚」を楽しむ

独自視点を確立する入り口として、「マイナーなもの・端に追いやられているもの」に光を当てる練習をする。

- 小津安二郎の映画をカメラアングルの低さだけ観察する
- 脇役に肩入れする
- 「仏像最高！」と言って仏像ブームを起こした、みうらじゅん的視点

こういう見方をエッセイで書く習慣をつけると、次第に柔軟なものの見方が身につく。

## 6. 引用と出典での「おトク感」

### 引用の核心的役割

自分の発見がつまらないときの「必殺技」が引用だ。ただしコピペではない。本を読んで気に入った一節を書き取り、出典を明記して文章に組み込む。

野村克也監督が好んで使う「行動を変えると習慣が変わる、習慣が変わると人格が変わる、人格が変わると運命が変わる」——出典はウィリアム・ジェームズだと明記すると、読者は「野村さんの言葉かと思ったらアメリカの心理学者の言葉なんだ」と知識が得られ、**おトク感**が生まれる。

### 引用のルール

| 良い引用 | 避けるべき引用 |
|---------|--------------|
| あまり知られていない言葉 | 手垢がついた名言・諺 |
| 出典を明記した言葉 | 出典不明の引用 |
| 本を読みながら自分でふと見つけた言葉 | 誰でも知っているような言葉（「急がば回れ」等） |
| 「ビッグネーム」からの引用（ニーチェ、フーコー等） | 小学校の道徳水準の諺 |

「引用ノート」を作る習慣を持つこと。引用した一節の横に「なぜこれに惹かれたか」を必ず書き加える。エピソードやインスピレーションと引用文を結びつけることができる。

### 「縛り」引用の練習法

「『論語』の一節を必ず組み込んでエッセイを書く」という制約を自分に課す。

制約があることで、書きにくいのではなく逆に書きやすくなる。「この一節に合うエピソードはどれか」という探し方になり、二つの要素が自然につながる。

### ビッグネームを「月のように」使う

自分で光を放てなくても、太陽（ビッグネーム）の光を反射して輝けばいい。

「この現代社会の閉塞状況を打ち破るためには、まさにニーチェの言うこの言葉が答えになっているのではないか」——多少強引でも、その「持っていき方の強引さ」が評価される。

## 7. 三段構えの論法を速く書く

### 文章の段取り（4ステップ）

| ステップ | 内容 | ポイント |
|---------|------|---------|
| ①最後の文章を決める | 結論となる一文を先に決定する | ゴールが決まれば、途中でフラフラしても戻れる |
| ②タイトルを決める | ゴールに対する「疑問文」の形にする | 「通勤地獄の解消の決め手はあやとり感覚か」 |
| ③通過地点を3つ決める | ネタ出し時の重要ワードから3つ選ぶ | 三段構えで読者は納得する |
| ④とにかく速く書く | 「多少無理があっても力技で進む」 | 書く速度は実力だ |

### 三段構えが効く理由

人は三段構えの組み立てに「へえ」と納得する。

「一がこれで」→「えー！」
「そして二がこれで」→「へぇ」
「結局、三なんですよ」→「ほう」

という具合に、3つのステップを踏んだものに説得力が生まれる。

### 速く書くための心構え

スタートとゴールを先に決め、迷いなく突き進む。多少論理が無理でも、ゴールに向かって「ガン、ガン、ガン」と力技で攻めていく練習が、書くスピードを格段に速くする。

書き終えた後の推敲で直せばいい。最初から完璧を目指す必要はない。

## 8. アウトプット優先の速読術

### 「書く」要請が読書を変える

「いつかアウトプットするかもしれない」という意識で読む。もっと直接的には「今これに対して書かなければならない」という要請があるときに、アウトプット優先で読む。

この姿勢に変えると：
- 自分と関係ないところをどんどん飛ばす
- 速読が自然にできるようになる
- 記憶・定着度が高まる

### 「縁」で本を選ぶ

読書は「縁」だ。渋谷の雑踏で「この人とは縁があるかも」と感じ取るのと同じように、本にも「縁の濃密度」がある。

- **Aランク**：赤い糸を感じる本。著者の人格ごと読み込む。
- **Bランク**：流れ次第でつきあう。
- **Cランク**：縁はない。バッサリ飛ばす。

勘違いも芸のうち。「これは運命かも」と感じた本との付き合い方を大切にする。

### 「濃淡読み」の実践

砂丘で砂金を探す漁師のように読む。

- **魚影のない場所（関係ないページ）** → パパッと目を通す
- **魚群探知器に反応があった場所** → じっくり読み込み、引用候補を探す

カフカ『城』のように退屈な場面が続いても、キラリと光る言葉に出会えれば「砂金を見つけた」ようなもの。一冊から「これでエッセイが一本書ける」という一節が取れれば上出来だ。

### 「保存期間の長いもの」を選ぶ

情報の「もちがいい」本とは、何年経っても古びない本だ。

- 著者の人格とセットで読む本は記憶に残る
- 時事ニュースをもとに書いた引用は、すぐに古くなる
- 古典・哲学・思想書は引用の保存期間が長い

アウトプット優先の視点から言えば、「ゆっくり味わいながら」ではなく「縁のあり・なし」を瞬間的に仕分けしながら読む。目次で章ごとの「縁」を見分けるのも有効だ。

### 齋藤孝メソッドの核心3原則

1. **発見がなければ文章ではない** — 上手い文章より、読み手のものの見方を変える文章を目指す
2. **凡庸さは恥である** — 一般論に乗っかる文章に甘んじない。「マイナー感覚」で独自視点を鍛える
3. **おトク感が文章を輝かせる** — 引用と出典で読者に「読んで得した」と感じさせる
