## なぜ参照するのか

正確なソース参照は以下の理由により必須である：

| 理由 | 説明 |
|------|------|
| **学術的礼儀** | 元の著者の知的貢献を認め、読者が原典を容易に見つけられるようにする |
| **信頼性の構築** | 権威ある情報源を参照していることを示し、記述内容への信頼を醸成する |
| **文脈化** | 自分のアイデアをより広い学術的フレームワークに位置づける |
| **知識の積み重ね** | 既存の文献を活用して自分の議論を構築していることを示す |
| **学術的誠実性** | 他者のアイデアを自分のものとして提示していない（plagiarism ではない）ことを証明する |
| **学術コミュニティへの参加** | 教育・研究分野の知識コミュニティの一員として位置づけられる |

## 参照システムの種類

主要な参照システムは以下の2種類：

| システム | 別名 | 特徴 |
|---------|------|------|
| **Harvard system** | name-date / author-date | 本文中に（著者名, 年）を記載。教育分野で広く採用 |
| **Vancouver system** | number / citation-sequence | 本文中に番号を記載 |

**教育分野では Harvard system が標準として使用される。**

### House style（出版スタイル）の違い

同じ Harvard system でも、出版社や大学ごとに細部の表記が異なる：

- カンマの有無（例: `(Thomas 2016)` vs `(Thomas, 2016)`）
- ページ番号の表記（例: `(Thomas, 2016, p. 36)` vs `(Thomas, 2016: 36)`）
- イニシャルの扱い（`Smith, J` vs `Smith, J.` vs `Smith, Jerome`）

**重要**: どのスタイルを使うかは大学や学科の指定に従う。一度選んだスタイルは文書全体で**一貫させる**こと。

## Harvard system の基本

### 基本構造

**本文中の引用**:
- 著者の姓（複数著者の場合はすべての姓）
- 出版年
- カンマで区切る（スタイルによってはカンマなし）

**文末の参考文献リスト**:
- 著者の姓のアルファベット順に並べる
- タイトル: "References" を使用

### 重要原則

**One-to-one match（一対一対応）**:
- 本文中で引用したソースはすべて参考文献リストに記載
- 参考文献リストに記載したソースはすべて本文中で引用されている

**"Bibliography" と "References" の違い**:
- **Bibliography**: 幅広い関連文献（直接引用していないものも含む）
- **References**: 本文中で実際に引用したソースのみ
- アカデミックライティングでは **"References"** を使用

## In-text citation テクニック

### 複数の参照を並べる

**原則**: 時系列順に並べ、同じ年のものはアルファベット順

```
(Li, 2002; James and Roberts, 2007; Jones, 2007)
```

複数参照はセミコロンで区切る。

### 同じ著者の同年複数文献

同じ著者が同じ年に複数の論文を出版している場合、年の後に `a`, `b`, `c` を付ける：

```
(Li, 2002a)
(Li, 2002b)
```

参考文献リストでも同様に：

```
Li, D (2002a) [タイトル1]. ...
Li, D (2002b) [タイトル2]. ...
```

### 直接引用（Direct quotations）

| 要素 | ルール |
|------|--------|
| **引用符** | シングル（'...'）またはダブル（"..."）で囲む。スタイルに従い一貫させる |
| **ページ番号** | 必須。例: `(Thomas, 2016, p. 36)` |
| **正確性** | 原文を**完全に正確に**コピーする |
| **省略** | 省略箇所には ellipsis (`...`) を使用。場合により `[...]` で囲む |
| **挿入・変更** | 文法調整のために挿入した語は角括弧 `[...]` で囲む |
| **長い引用** | ダブルインデント、または小さいフォントサイズで表記 |

#### 例: 長い引用

```
According to Smith (2003, p 185):

> A good focus group is more than the sum of its parts. The researcher
> harnesses the group process, encouraging the group to work together to
> generate more in-depth data based on interaction [...]. [He or she] helps
> the group create a reflective environment in which the group can take an
> issue, approach it as they choose and explore it fully.
```

#### 例: 短い引用

```
Smith (2003) sees a good focus group as 'more than the sum of its parts' (p 185).
```

**注意**: 引用を使いすぎるのは避ける。引用の乱用は批判的思考の欠如や plagiarism の兆候となる。特に強力な表現や印象的な文章のみを引用すること。

### 'et al' の使用

**意味**: ラテン語の "et alii"（「その他」）

**使用条件**: 著者が**3名以上**の場合、本文中では最初の著者名のみを記載し、残りを "et al" で省略

```
(Johnson et al, 2003)
```

**重要**: 参考文献リストでは**全著者名を記載**する（house style で指定がない限り）。

## 参考文献リストの作成

### 主要な文献タイプ

| タイプ | 説明 |
|--------|------|
| **書籍** | 単著または共著の著作 |
| **編集書籍の章** | 複数著者が各章を書き、編者がまとめた書籍の1章 |
| **ジャーナル論文** | 学術誌に掲載された論文 |

### 書籍のフォーマット

**基本形式**:

```
Surname(s) of author(s), Initial(s) (Year) Title: Subtitle. Edition [if 2nd or greater]. Place of publication: Publisher.
```

**例**:

```
Beck, I (2005) Bringing Words to Life. New York: Guilford Press.
Crystal, D (2005) How Language Works. London: Penguin.
Jolliffe, W and Waugh, D with Carss, A (2015) Teaching Systematic Synthetic Phonics in Primary Schools. 2nd ed. London: Learning Matters/SAGE.
```

### 編集書籍のフォーマット

**書籍全体を引用する場合**:

```
Surname(s) of editor(s), Initial(s) (ed/eds) (Year) Title: Subtitle. Place of publication: Publisher.
```

**例**:

```
Brock, C (ed) (2015) Education in the United Kingdom. London: Bloomsbury.
Jones, D and Hodson, P (eds) (2017) Unlocking Speaking and Listening: Developing Spoken Language in the Primary Classroom. 3rd ed. London: David Fulton.
```

**編集書籍の特定の章を引用する場合**:

```
Surname(s) of chapter author(s), Initial(s) (Year) Title of Chapter. In Surname(s) of editor(s), Initial(s) (ed/eds) Title of Book: Subtitle. Edition. Place of publication: Publisher.
```

**例**:

```
English, E (2017) Is There a Place for Drama? In Waugh, D, Jolliffe, W and Allott, K (eds) Primary English for Trainee Teachers. London: Sage.
```

### ジャーナル論文のフォーマット

**基本形式**:

```
Surname(s) of author(s), Initial(s) (Year) Title of Article. Journal Title, Volume(Part), Pages.
```

**例**:

```
Apthorp, H (2006) Effects of a Supplemental Vocabulary Programme in Third Grade Reading/Language Arts. Journal of Educational Research, 100(2), 67-79.
```

**DOI（Digital Object Identifier）の扱い**:

DOI は論文の恒久的なアドレスを示す。以下のいずれの形式も許容される：

```
[論文情報]. DOI: 10.xxxx/xxxxx
```

または

```
[論文情報]. Available at: [URL] (accessed [日付]).
```

### その他の文献タイプ

#### 修士論文・博士論文

```
Surname, Initial (Year) Title. [type of thesis], [location: university name].
```

**例**:

```
Waugh, D (2000) Primary Schools at the Crossroads: A Study of Primary Schools' Abilities to Implement Educational Change. Unpublished PhD thesis. Hull: University of Hull.
```

#### 新聞・雑誌記事

```
Surname, Initial (Year) Title. Publication Name, Full Date, [online] Available at: [URL] (accessed [日付]).
```

#### 組織・団体著者

機関名を著者名として使用：

```
DfE (2011) Teachers' Standards: Statutory Guidance for School Leaders, School Staff and Governing Bodies. London: DFE 00066-2011.
```

**略称使用例**:
- DfE（Department for Education）
- Ofsted（Office for Standards in Education）
- WHO（World Health Organization）

#### 電子ソース

**追加要素**:
1. メディアタイプ（`[online]`, `[PDF]`, `[DVD]` 等）
2. "Available at: [URL]" または "Available from: [source]"
3. アクセス日（`(accessed [日付])`）

**例**:

```
Merriam-Webster Dictionary, [online] Available at: www.merriam-webster.com (accessed 15 May 2017).
```

## よくある問題と解決策

### 匿名著者

辞書や法律文書など、著者が不明な場合は**文書タイトル**で参照：

```
Collins English Dictionary (2011) 11th ed. Glasgow: William Collins & Sons.
Education Reform Act (1988) London: HMSO.
```

### 二次引用（Secondary citations）

**原則**: できる限り原典（primary source）を直接参照する。

やむを得ず二次引用する場合は "cited in" または "cited by" を使用：

**本文中**:

```
(Nash, 1978, cited in Galton and Patrick, 1990, p 9)
```

**参考文献リストには読んだ文献（二次ソース）のみを記載**:

```
Galton, M and Patrick, H (eds) (1990) Curriculum Provision in the Small Primary School. London: Routledge.
```

### スタイルのバリエーション

| 要素 | バリエーション例 |
|------|----------------|
| **本文中カンマ** | `(Thomas 2016)` vs `(Thomas, 2016)` |
| **ページ番号** | `(Thomas, 2016, p. 36)` vs `(Thomas, 2016: 36)` |
| **et al 表記** | `et al` vs `et al.` vs *et al.* |
| **イニシャル** | `Smith, J` vs `Smith, J.` vs `Smith, Jerome` |
| **ed/eds 表記** | `ed` vs `ed.` vs `eds` vs `eds.` |
| **タイトル大文字** | 小文字、すべて大文字、先頭大文字など複数の方式がある |

**重要**: 選んだスタイルを**一貫して**使用すること。

### 参照ソフトウェア

| ツール | 特徴 |
|--------|------|
| **EndNote** | 商用。高機能で多機能 |
| **Mendeley** | 無料版あり。PDF管理機能 |
| **Zotero** | オープンソース。ブラウザ拡張機能 |
| **Papyrus** | 商用。ワードプロセッサ統合 |

**メリット**:
- 本文引用と参考文献リストの一対一対応を自動保証
- スタイル（Harvard / APA / Vancouver 等）の自動切替

**デメリット**:
- 有料ソフトの場合、個人購入が必要
- ワードプロセッサとの統合でフォーマットの不具合が起きることがある

## 批判的なソース活用

### Uncritical vs Critical な使い方

| Uncritical（非批判的） | Critical（批判的） |
|----------------------|------------------|
| 他者の声が文章を支配 | 他者の引用は自分の声を**支持**するが、置き換えない |
| 文献の内容を**描写**するだけ（knowledge-telling） | 内容を**分析・評価**し、知識を変容させる（knowledge-transforming） |
| パラフレーズで単語を同義語に置き換えるだけ | 真の理解を示し、自分の議論に統合する |
| 引用に過度に依存し、引用自体が意味を語ると考える | 引用を**説明**し、自分の議論に統合する |
| ソースを「パッチワーク」状に並べるだけ | ソースを**比較・対照**し、共通点や相違点を示す |
| ランダムな順序でソースを提示 | 文献の**発展・パターン・関係性**を特定し、配置を決める |
| 読者が文献と課題の関連を推測することを期待 | 文献と課題の関連を**明示的に説明**する |

### Originality（独自性）の誤解

**誤解**: 独自性とは、完全に新しい理論やアイデアを自分で作り出すこと

**正しい理解**: 既存の文献に基づきながら、トピックを**自分独自の方法**で扱うこと

- 既存の視点と証拠を評価し、納得したものに基づいて自分のアイデアを構築
- 文献を土台としつつ、**自分の声**を最も強く保つ

### Author-prominent vs Information-prominent

**Author-prominent（著者を強調）**:

著者名が文の構造の一部になる：

```
Butler (2007) questions the effectiveness of drugs, claiming that scientific accuracy is being sacrificed to marketing spin.
```

**情報を強調する場合**: 著者名は括弧内のみに記載：

```
Scientific accuracy may be sacrificed to marketing spin in drug marketing (Butler, 2007).
```

**使い分け基準**:

| 使用場面 | 推奨形式 | 理由 |
|---------|---------|------|
| 重要な論争や議論の主要人物を強調 | Author-prominent | **誰が**言っているかが重要 |
| 情報の内容自体を強調 | Information-prominent | **何が**言われているかが重要 |

### 著者を示す動詞の選択

パラフレーズ時に使う動詞は、著者のアイデアに対する自分の解釈を伝える：

| 動詞 | 用途 |
|------|------|
| **acknowledge** | 制約や限界を認める場合 |
| **advocate** | 強く推奨している場合 |
| **argue that** | 主張や論証をしている場合 |
| **identify** | データから主要テーマを見出した場合 |
| **define something as** | 概念を定義している場合 |
| **distinguish between** | 2つ以上の概念を区別している場合 |
| **point out** | 重要な点を指摘している場合 |

**例**:

```
Whiteman et al (2016) advocate targeted training for teaching assistants.
Barwick et al (2011) identified four principal themes.
Broadbent (2015) distinguishes between mild and severe forms of dyslexia.
```

## Academic malpractice の回避

### Academic malpractice の種類

| 種類 | 定義 |
|------|------|
| **Plagiarism** | 他者のアイデア・著作・言葉を明確な謝辞なしに使用 |
| **Collusion** | 自分の成果物への他者の貢献を隠蔽 |
| **Falsification** | 結果の捏造や改ざん |

### Plagiarism のリスク

- 多くの大学が **Turnitin** 等のソフトウェアで論文をスクリーン
- 文献や他学生のエッセイとの一致が発見されると厳密に審査
- 重大または繰り返しの違反は**ゼロ点**、再提出不可、懲戒処分の対象

**教職課程の学生への影響**:
- 教員スタンダードは**高い個人的・職業的行動基準**を要求
- 故意の academic malpractice は「良い品性」への障害と見なされる
- 懲戒委員会への出席、最悪の場合は大学・教職課程からの退学

### Plagiarism 回避のベストプラクティス

| 原則 | 実践方法 |
|------|---------|
| **すべての情報源を明示** | 直接引用でもパラフレーズでも、すべてのアイデアと情報の出典を記載 |
| **引用の過度な使用を避ける** | 特に印象的な表現のみを引用。全文ではなく**効果的な語句**を統合 |
| **文献を説明・解釈する** | ソースを自分の議論に統合し、自分の執筆目的に合わせて使用 |
| **パッチワークを避ける** | ソースを単に並べるのではなく、関連性を示し、自分の声で統合 |

**重要**: 引用は悪いことではない。適切に使えば、研究の深さと独自性を**強化**する。

## まとめ

正確で効果的なソース参照は、アカデミックライティングの基礎である。

**実践のポイント**:

1. **Harvard system のルールを正確に適用**（大学のスタイルガイドに従う）
2. **批判的にソースを活用**し、自分の議論を構築する土台とする
3. **一貫性**を保つ（選んだスタイルを文書全体で統一）
4. **参照ソフトウェア**の活用を検討（大規模な文献管理に有効）
5. **Plagiarism を避ける**ため、すべてのソースを明示し、批判的に統合する

引用と参照は、学術コミュニティへの参加を示し、自分のアイデアの信頼性と独自性を高める重要な手段である。
